# Presentify

- **URL**: https://presentifyapp.com
- **一句话**: macOS 菜单栏工具，让你在屏幕上实时画圈标注、高亮光标、放大局部，直播和录屏更清晰
- **详细说明**: Presentify 允许用户在任何屏幕内容上用笔、荧光笔、形状和文字即时标注，同时提供光标光晕效果、聚光灯和缩放功能，帮助观众快速跟上演讲节奏。轻量菜单栏常驻，不干扰正常工作流，特别适合教师、YouTube 创作者、技术演讲者在直播或演示中增强表达清晰度。
- **目标用户**: 教师、内容创作者、技术讲师、销售演示者、直播主
- **视觉风格**: 实用工具，macOS 原生风格，菜单栏图标
- **主色调**: 系统默认，强调色可自定义
- **氛围关键词**: 演示、标注、教学、直播、macOS工具
- **可归类方向**:
  - 常规: 演示与录屏工具
  - 创意: make-it-beautiful / keyboard-people
