# Infracost

- **URL**: https://cost.dev
- **一句话**: 把云成本意识嵌入 AI 编程代理和 IDE 的开发者工具，在部署前看清每行 Terraform 代码的价格
- **详细说明**: Infracost 集成 1000+ 项 AWS/Azure/GCP 服务的实时定价，让开发者在写基础设施代码时直接对比不同配置的成本差异。支持 Claude、GitHub Copilot、Cursor 等 AI 代码助手，以及 VS Code、JetBrains、Neovim 等主流 IDE，并能自动修复整个代码库的 tagging 合规问题。个人免费，企业版支持跨团队策略执行和自定义定价协议。
- **目标用户**: 使用 Terraform/CloudFormation/AWS CDK 的开发者，平台工程师，使用 AI 编程助手的工程团队
- **视觉风格**: 开发者工具风，代码集成界面，简洁文档导向
- **主色调**: 深色，绿色强调（成本可视化）
- **氛围关键词**: 云成本、FinOps、开发者工具、AI代理、基础设施
- **可归类方向**:
  - 常规: 云成本管理 / 开发者工具
  - 创意: solo-founder-back-office / keyboard-people
