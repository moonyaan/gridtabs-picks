# Freu AI

- **URL**: https://freu.ai
- **一句话**: Mac 桌面 AI 代理，用自然语言自动化任意 macOS 应用操作，学会后零 Token 重复执行
- **详细说明**: Freu AI 通过语义 UI 识别学习你的操作流程，编译为确定性 DSL 命令，之后每次执行无需再调用云端 AI，大幅降低重复自动化成本。支持自动化 Spotify、Slack、Figma 等任意 Mac 应用，无需写代码或固定坐标。
- **目标用户**: 有重复性桌面操作需求的 Mac 用户、AI 工具爱好者、效率极客
- **视觉风格**: 简洁，加载后内容极简，强调"低成本自动化"核心卖点
- **主色调**: 不明显（极简单色界面）
- **氛围关键词**: Mac自动化、零成本重复执行、语义UI、本地运行
- **可归类方向**:
  - 常规: macOS 工具、AI 自动化
  - 创意: swiss-army-knife-sites、focused-life
