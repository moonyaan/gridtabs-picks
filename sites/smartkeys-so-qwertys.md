# QWERTYS

- **URL**: https://smartkeys.so/qwertys
- **一句话**: 键盘主题俄罗斯方块——让字母键落到正确位置才能拼出完整键盘
- **详细说明**: QWERTYS 是一个融合了俄罗斯方块机制的键盘主题益智游戏。键盘上的字母键从屏幕顶部落下，玩家需要拖动每个键让它落在 QWERTY 键盘布局中正确的位置。放错的键变成"垃圾"需要炸弹清除，堆满屏幕则游戏结束。有多个难度等级和排行榜，键盘知识+游戏技巧双重考验。
- **目标用户**: 键盘爱好者、打字练习者、休闲游戏玩家
- **视觉风格**: 现代扁平设计，键帽像素感
- **主色调**: 深色 / 白色键帽
- **氛围关键词**: 游戏、键盘、益智、趣味、排行榜
- **可归类方向**:
  - 常规: internet-curiosity-museum
  - 创意: keyboard-people、pixel-and-handmade
