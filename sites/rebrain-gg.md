# Rebrain.gg

- **URL**: https://rebrain.gg
- **一句话**: 用社区对话替代刷短视频的学习平台，和其他人一起讨论真正感兴趣的话题
- **详细说明**: Rebrain.gg 是一个以"学习"为核心的社区平台，用户围绕编程、语言、技能等话题发起协作式学习对话。设计理念是给无意义刷屏找一个有意义的替代出口——让大脑的时间被值得的事情占据。
- **目标用户**: 自学者、技能爱好者、想脱离刷屏习惯的人
- **视觉风格**: 现代 Web 应用，支持亮色/暗色切换，对话流式布局
- **主色调**: 双主题（亮/暗）
- **氛围关键词**: 协作、社区、学习导向、对话式、有意义
- **可归类方向**:
  - 常规: 学习社区、知识分享、自我提升
  - 创意: 反算法、用学习对抗刷屏、大脑健身房
