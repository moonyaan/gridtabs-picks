# Kyushu

- **URL**: https://kyushu.dev
- **一句话**: 把 JavaScript/TypeScript 编译成独立 WASM 二进制的自托管 JS 执行沙箱
- **详细说明**: Kyushu 是一个开源 CLI 工具，让开发者把 JS/TS 处理函数编译成独立的 WebAssembly 二进制文件，然后用单个 `kyu` 命令在任何地方运行——无需 Node.js、Bun 或 Docker。采用 Cloudflare Workers 风格的 fetch handler API，专为服务器环境设计，单文件即可部署。
- **目标用户**: 后端开发者、DevOps 工程师、追求轻量部署的独立开发者
- **视觉风格**: 极简开发者文档风格
- **主色调**: 白色 / 深色代码块
- **氛围关键词**: 轻量、可移植、WASM、无依赖、serverless
- **可归类方向**:
  - 常规: keyboard-people、self-hosted-gems
  - 创意: swiss-army-knife-sites
