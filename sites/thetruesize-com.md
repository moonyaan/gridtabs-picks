# The True Size Of

- **URL**: https://www.thetruesize.com
- **一句话**: 把任何国家拖到地图不同位置，直观看清它的真实面积，彻底打破墨卡托投影造成的大小错觉
- **详细说明**: 标准世界地图因墨卡托投影而严重扭曲高纬度地区的面积，格陵兰岛看起来比非洲还大，实则差了约14倍。The True Size Of 让你自由拖动国家轮廓进行纬度对比，用互动的方式还原地理真相。
- **目标用户**: 地理爱好者、学生、好奇心旺盛的普通用户
- **视觉风格**: 地图风，简洁交互，高对比度国家轮廓叠加
- **主色调**: 深蓝地图色 + 高亮对比色
- **氛围关键词**: 地理、认知颠覆、地图、可视化、真相
- **可归类方向**:
  - 常规: 地理教育 / 数据可视化
  - 创意: 你以为的世界地图，其实是一个谎言
