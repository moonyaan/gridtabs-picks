# Oku

- **URL**: https://oku.io
- **一句话**: 把 Reddit、YouTube、HN、RSS 聚合到一个无算法仪表板的信息过滤工具
- **详细说明**: Oku 是个人信息聚合平台，将 Reddit、YouTube、Hacker News、RSS 源等内容整合到单一仪表板，无广告、无弹窗、无算法推荐。支持智能摘要、内容过滤和自动每日/周刊文摘，帮助信息工作者在几分钟内浏览完所有重要内容。
- **目标用户**: 信息工作者、研究人员、希望减少信息噪音的知识工作者
- **视觉风格**: 极简现代设计，卡片式布局，大量留白，深浅主题自适应
- **主色调**: 珊瑚橙 (#d97757)，浅灰背景，深灰背景（深色模式）
- **氛围关键词**: 清晰、专注、高效、极简、去除干扰
- **可归类方向**:
  - 常规: 信息聚合 / RSS 阅读器 / 内容管理
  - 创意: always-on-internet / focused-life
