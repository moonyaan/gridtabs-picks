# Blogosphere

- **URL**: https://text.blogosphere.app
- **一句话**: 独立博客聚合器，按时间顺序展示个人博主的最新文章，找回独立 Web 的阅读感
- **详细说明**: Blogosphere 是一个收录独立博主内容的聚合平台，涵盖艺术、科技、摄影、个人写作等多种主题，以时间顺序呈现内容，完全无算法干预。内置随机"漫游"功能，帮助读者发现平时不会遇到的优质个人博客，是对抗平台化内容的精神角落。
- **目标用户**: 热爱阅读的人、个人博主、厌倦算法推荐内容的读者
- **视觉风格**: 极简文字优先，列表排版，复古博客美学
- **主色调**: 白色、深灰、黑色
- **氛围关键词**: indie-web, blogs, no-algorithm, reading, discovery
- **可归类方向**:
  - 常规: 内容发现、博客聚合
  - 创意: no-algorithm-sites、always-on-internet
