# Visdiff

- **URL**: https://visdiff.com
- **一句话**: AI 生成代码 + 自动视觉校准，让 Figma 设计稿和实际代码真正对齐
- **详细说明**: Visdiff 是一个坐落在 Figma 和代码库之间的视觉差异引擎。AI 生成代码后，Visdiff 截图、像素级对比设计稿，发现偏差后自动修正，直到完全匹配。支持多断点、MCP 协议接入，彻底解决 AI 写出来的 UI "差不多对但就是不对" 的问题。
- **目标用户**: 使用 AI 编程工具的前端开发者、设计转开发团队、数字代理公司
- **视觉风格**: 极简现代，衬线标题字体配无衬线界面，带动画光标和故障艺术效果的 logo
- **主色调**: 深色背景、亮色强调色（用于发光动画）、红色对比色
- **氛围关键词**: precise、AI-native、developer-tool、visual-diff、pixel-perfect
- **可归类方向**:
  - 常规: AI开发工具、设计转代码、前端工作流
  - 创意: keyboard-people、ai-no-code-2026
