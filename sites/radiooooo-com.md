# Radiooooo

- **URL**: https://radiooooo.com
- **一句话**: 音乐时光机，在世界地图上选择国家和年代，聆听那个地方那个时代的真实广播和音乐
- **详细说明**: Radiooooo 是一个独特的音乐探索工具，让你在交互式世界地图上选择任意国家和任意历史年代（从 1900 年代至今），然后播放来自那个地方那个时代的真实音乐。想听 1970 年代的巴西、1950 年代的日本或 1980 年代的尼日利亚音乐？只需点击地图即可。是音乐探索与历史旅行的完美结合。
- **目标用户**: 世界音乐爱好者、历史文化探索者、DJ、乐迷、好奇心强的普通用户
- **视觉风格**: 复古华丽，地球仪/地图风格界面，带有老式广播的质感
- **主色调**: 深色背景、暖金、复古地图棕
- **氛围关键词**: 时光机、世界音乐、探索、复古、文化旅行
- **可归类方向**:
  - 常规: 音乐发现 / 全球广播
  - 创意: 声音时光机 / 跨越时空的耳朵
