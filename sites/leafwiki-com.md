# LeafWiki

- **URL**: https://leafwiki.com
- **一句话**: 极轻量的自托管个人 Wiki，单文件部署，Markdown 优先，完全无数据库依赖
- **详细说明**: LeafWiki 是为个人用户设计的极简 Wiki 工具，单文件即可部署，所有内容以 Markdown 格式存储为纯文本文件，无需配置数据库或复杂的服务器环境。适合想要轻量级个人知识管理的开发者和技术爱好者。
- **目标用户**: 独立开发者、自托管爱好者、需要简单个人知识库的技术用户
- **视觉风格**: 极简，内容优先，朴素的 Markdown 渲染风格
- **主色调**: 白色 + 绿色点缀
- **氛围关键词**: 极简、自托管、Markdown、个人知识库、无数据库
- **可归类方向**:
  - 常规: 个人知识管理 / 自托管工具
  - 创意: 最轻的 Wiki——一个文件，装下你的全部知识
