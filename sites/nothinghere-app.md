# NothingHere

- **URL**: https://solee0524.github.io/NothingHere/
- **一句话**: macOS 隐私紧急按钮：一键隐藏所有窗口、静音、打开伪装文件，免费开源。
- **详细说明**: NothingHere 是一个极轻量的 macOS 菜单栏工具，按下自定义热键后同时执行三个动作：隐藏所有窗口、关闭系统声音、打开你预先设置好的"正经工作"文件。完全免费、开源，仅 5.8MB，需要 macOS 15.0+。
- **目标用户**: 所有在公共场合或办公室使用 Mac 的用户，尤其是经常切换私人和工作内容的人
- **视觉风格**: 极简落地页，菜单栏工具风格，干净直接
- **主色调**: 简洁白/黑，功能性为主
- **氛围关键词**: 隐私、极简、实用、开源、Mac工具
- **可归类方向**:
  - 常规: macOS工具 / 隐私保护
  - 创意: 独立开发者小工具 / 日常痛点解决
