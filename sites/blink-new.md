# Blink

- **URL**: https://blink.new
- **一句话**: 用 AI 对话几分钟内构建含后端、数据库和部署的完整 Web 应用
- **详细说明**: Blink 是全栈 AI 应用构建器，通过自然语言描述即可生成带数据库、用户认证、边缘函数和支付集成的生产级 React 应用。AI 代理自主设置基础设施、修复 bug，无需手动配置，并可一键部署到自定义域名。支持 GPT-5、Gemini、Claude 等 180+ AI 模型。
- **目标用户**: 非技术创业者、独立开发者、快速验证想法的产品经理
- **视觉风格**: 现代极简深色界面，干净的组件布局
- **主色调**: 纯黑背景，白色文字，灰色边框辅助
- **氛围关键词**: 快速、赋能、生产就绪、直觉、野心
- **可归类方向**:
  - 常规: AI 应用构建器 / No-Code 平台
  - 创意: Vibe Coding / 创业加速器
