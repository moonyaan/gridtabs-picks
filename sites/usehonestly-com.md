# Honestly

- **URL**: https://usehonestly.com
- **一句话**: 匿名团队反馈工具，用结构化问卷帮团队收集真实的内部意见，说真话不再有顾虑
- **详细说明**: Honestly 通过匿名化问卷和脉冲调研，帮助团队管理者了解真实的员工心声和团队状态。结构化的问题设计和数据汇总，让反馈文化的落地变得简单有据可查。
- **目标用户**: 团队管理者、HR、希望建立反馈文化的小型团队
- **视觉风格**: 简洁专业，信任感强，清晰的数据展示界面
- **主色调**: 深蓝 + 浅灰
- **氛围关键词**: 匿名、反馈、团队、诚实、管理
- **可归类方向**:
  - 常规: 团队管理工具 / 员工反馈
  - 创意: 让真话有地方说——匿名是团队健康最廉价的解药
