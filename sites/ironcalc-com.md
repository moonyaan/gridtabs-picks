# IronCalc

- **URL**: https://www.ironcalc.com
- **一句话**: Rust 构建的开源电子表格引擎，可嵌入应用或自托管部署，兼容 Excel
- **详细说明**: IronCalc 是一个用 Rust 构建的开源电子表格引擎和生态系统，支持 Excel 兼容性、多语言和自托管部署。可作为独立应用使用，也可嵌入 SaaS 产品中进行无外部依赖的表格逻辑处理。获得欧盟地平线计划和 NLnet 基金会资助，致力于成为"面向所有人的高级电子表格解决方案"。
- **目标用户**: 需要可嵌入电子表格方案的开发者、需要数据自主权的组织、SaaS 公司
- **视觉风格**: 极简现代技术设计，干净排版，大量留白，专业感
- **主色调**: 深灰/黑色主色，蓝色/青色强调
- **氛围关键词**: 开源、透明、可靠、轻量、高性能、隐私优先
- **可归类方向**:
  - 常规: 开发者工具 / 数据处理 / 开源项目
  - 创意: self-hosted-gems / keyboard-people
