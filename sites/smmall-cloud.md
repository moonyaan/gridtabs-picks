# Smmall Cloud

- **URL**: https://smmall.cloud
- **一句话**: 个人托管平台，用最小化服务器跑你的应用，专注小而精的理念
- **详细说明**: 一个定位于"小而精"的个人应用托管平台，帮助个人开发者用最少的资源部署和运行自己的应用，反对过度工程化，拥抱简单可靠的基础设施理念。
- **目标用户**: 独立开发者、个人项目作者、讨厌复杂云平台的工程师
- **视觉风格**: 简约，反复杂主义
- **主色调**: 白色/绿色或蓝色
- **氛围关键词**: 小而美、个人、极简、托管、独立
- **可归类方向**:
  - 常规: 云托管、开发工具
  - 创意: 够用就好的哲学
