# Ask Ellie

- **URL**: https://askellie.ai
- **一句话**: 在 Slack 中直接聚合 GitHub、Jira、Linear 工程上下文的 AI 工程助手
- **详细说明**: Ask Ellie 是面向工程团队的 Slack AI 助手，自动将 GitHub Issues、Jira Ticket、Linear Task 等分散的工程上下文整合到 Slack 中，开发者无需切换工具即可查询项目状态、工单信息和代码变更。2026 年 2 月 Product Hunt 热门发布。
- **目标用户**: 使用 Slack 的工程团队、需要减少工具切换成本的软件开发团队
- **视觉风格**: 推断为 SaaS 工具标准风格，简洁专业
- **主色调**: 未知
- **氛围关键词**: 工程效率、上下文聚合、团队协作、减少切换
- **可归类方向**:
  - 常规: 开发者工具、团队协作、AI 助手
  - 创意: 「Slack 里的工程大脑」
