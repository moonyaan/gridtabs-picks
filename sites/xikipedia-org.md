# Xikipedia

- **URL**: https://xikipedia.org
- **一句话**: 维基百科内容变成可刷的信息流，本地推荐算法
- **详细说明**: Xikipedia 将简单维基百科内容以社交媒体信息流形式呈现。用户可以浏览、点赞文章，算法会快速学习用户兴趣推荐相似内容。核心特色是隐私优先：算法完全在本地运行，数据不离开设备。支持完全离线使用、PWA 安装、多用户配置文件。展示简单算法即可学习用户偏好，无需复杂机器学习。
- **目标用户**: 对维基百科内容感兴趣的成年用户、关注隐私保护的用户、想了解推荐算法工作原理的技术爱好者
- **视觉风格**: 现代极简社交媒体风格，支持亮/暗色模式自动切换。主题色为蓝色(#2cafff)，暗色背景#15202B，边框色#38444D。采用卡片式布局、大量圆角设计(按钮圆角达64px)、系统字体、底部导航栏、SVG图标。
- **主色调**: 蓝色(#2cafff)、暗色背景(#15202B)
- **氛围关键词**: 隐私、本地算法、信息流、离线、极简
- **可归类方向**:
  - 常规：知识工具、维基百科、推荐算法、隐私工具、PWA应用
  - 创意：本地算法、信息流革命、隐私优先、维基百科新生、可刷的知识、离线体验
