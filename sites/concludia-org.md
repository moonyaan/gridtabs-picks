# Concludia

- **URL**: https://concludia.org
- **一句话**: 图形化论证探索平台，通过可视化逻辑图帮助你梳理和检验任何结论背后的推理
- **详细说明**: Concludia 是一个帮助用户构建和检验逻辑论证的可视化工具，通过图形化的方式展示论据之间的充分性和必要性关系。用户可以分解任何复杂的推理链条，识别前提漏洞，或与他人协作探索争议性结论。适合批判性思考训练、哲学讨论或复杂决策分析。
- **目标用户**: 哲学爱好者、批判性思考学习者、学术研究者、需要清晰推理的决策者
- **视觉风格**: 简约学术风，暖米白背景配深蓝点缀，衬线字体增加智识感
- **主色调**: 暖米白 (#f6f1e7)、深蓝 (#2b4a7e)、奶白
- **氛围关键词**: 分析性、智识、逻辑、思辨、协作推理
- **可归类方向**:
  - 常规: 思维工具 / 论证分析
  - 创意: 逻辑可视化 / 思想地图
