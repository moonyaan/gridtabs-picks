# Krisp

- **URL**: https://krisp.ai
- **一句话**: 实时噪音消除 + 口音转换 + AI 会议记录，让远程会议听起来像在录音棚里开的。
- **详细说明**: Krisp 将三项能力合一：业界顶级的实时降噪（消除背景噪音、回声和串音）、AI 口音转换（实时调整说话者口音提升可懂度）、AI 会议助手（自动转录、生成摘要和行动项）。支持 SOC 2 / GDPR / HIPAA 合规，有面向开发者的 Voice SDK 和呼叫中心专用版本。
- **目标用户**: 远程工作者、呼叫中心团队、内容创作者、需要高质量远程会议体验的专业人士
- **视觉风格**: 现代科技 SaaS 风，紫蓝渐变，大量留白，专业中带亲切
- **主色调**: 紫色、蓝色渐变
- **氛围关键词**: 清晰度、远程工作、AI音频、专业、信任感
- **可归类方向**:
  - 常规: 会议工具 / AI音频处理
  - 创意: 声音可能性 / 远程工作基础设施
