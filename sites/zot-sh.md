# Zot

- **URL**: https://www.zot.sh
- **一句话**: 单一静态 Go 二进制的轻量编程代理，连接 25+ LLM 供应商，自带文件读写、Shell 执行和 Telegram 桥接。
- **详细说明**: Zot 是一款零依赖的终端 AI 编程代理，以单个静态 Go 二进制形式发布，无需 Docker 或运行时环境。它支持 Anthropic、OpenAI、DeepSeek、Gemini 等 25+ 个 LLM 提供商，提供文件读写、Shell 命令执行等核心工具，内置全功能 TUI 界面（流式输出、slash 命令、内联侧边聊天），还支持任意语言扩展和 Telegram 消息桥。设计哲学是"小到能完全理解，真实到能学习代理系统如何工作"。
- **目标用户**: 开发者、想了解 AI 代理内部机制的工程师、偏好轻量工具的终端用户
- **视觉风格**: 极简技术风格，以终端界面截图为主
- **主色调**: 深色终端风格，中性色
- **氛围关键词**: 轻量、零依赖、多模型、开源、终端友好
- **可归类方向**:
  - 常规: 开发者工具/AI编程代理
  - 创意: 一个二进制文件，连接整个 AI 宇宙
