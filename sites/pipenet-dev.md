# Pipenet

- **URL**: https://pipenet.dev
- **一句话**: 现代开源隧道工具，将本地服务暴露到互联网，Localtunnel 的替代品
- **详细说明**: Pipenet 是 Localtunnel 的现代化开源替代方案，客户端和服务端一体打包，可以快速将本地开发服务器暴露到公网，方便 webhook 测试、演示和协作开发。自托管友好，配置简单，是开发者在本地调试外部服务的实用基础工具。
- **目标用户**: 开发者、需要本地服务公网访问的工程师
- **视觉风格**: 极简工具型，命令行文档风格
- **主色调**: 深色，终端感
- **氛围关键词**: 开发者工具、隧道、本地暴露、开源、自托管
- **可归类方向**:
  - 常规: 开发者工具 / 网络工具
  - 创意: 本地开发的对外窗口
