# Sure.is ANSI Art Viewer

- **URL**: https://sure.is/ansi/
- **一句话**: 浏览器里重现拨号时代的 ANSI 艺术——支持模拟波特率播放，全屏滚动，拖拽即看。
- **详细说明**: Sure.is 是一个纯浏览器 ANSI 艺术查看器，支持 .ANS、.BIN 格式和 .ZIP 艺术包，可调节字体（8×16 或 9×16）、缩放图案，还能模拟 300 到 14400 的拨号速率播放，完整复现 90 年代 BBS 文化的视觉体验。
- **目标用户**: 复古计算机爱好者、BBS 文化研究者、数字艺术收藏者、网络怀旧人群
- **视觉风格**: 刻意复古，终端等宽字体，模拟 CRT 显示器的 ANSI 字符画界面
- **主色调**: 黑色背景配彩色 ANSI 字符，经典终端色彩
- **氛围关键词**: 怀旧、BBS文化、复古计算机、互联网考古、像素艺术
- **可归类方向**:
  - 常规: 数字文物 / 艺术查看器
  - 创意: 互联网好奇心博物馆 / 无算法场所
