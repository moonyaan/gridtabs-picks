# Bored Spreadsheet

- **URL**: https://boredspreadsheet.com/workbook
- **一句话**: 把经典小游戏伪装成无聊工作表格，办公室摸鱼神器
- **详细说明**: Bored Spreadsheet 把扫雷、2048、纸牌等经典游戏完美伪装成 Excel 风格的"运营工作簿"。界面有真实的菜单栏和表格样式，游戏名称全部替换成商务术语——扫雷叫"风险审计格"，2048 叫"单元格合并模型"。提供个人最佳记录和周排行榜。
- **目标用户**: 上班族、需要在严肃环境中偷偷放松的人
- **视觉风格**: 逼真 Excel/电子表格 UI 风格，绿色/灰色配色
- **主色调**: 绿色 / 白色 / 灰色
- **氛围关键词**: 幽默、伪装、办公室、解压、彩蛋
- **可归类方向**:
  - 常规: internet-curiosity-museum
  - 创意: no-algorithm-sites、pixel-and-handmade
