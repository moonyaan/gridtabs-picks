# PdfDing

- **URL**: https://pdfding.com
- **一句话**: 自托管 PDF 管理平台，支持标注、书签和分享，数据完全自主掌控
- **详细说明**: 一个可以自行部署的 PDF 管理平台，支持 PDF 标注、书签、高亮和分享功能，所有数据存储在自己的服务器上，无需担心隐私问题。适合有大量 PDF 文件需要管理的用户。
- **目标用户**: 注重数据隐私的用户、研究人员、知识工作者、自托管爱好者
- **视觉风格**: 简洁实用，专注内容
- **主色调**: 白色/橙色或红色点缀
- **氛围关键词**: 自托管、隐私、PDF、管理、掌控
- **可归类方向**:
  - 常规: 文档管理、PDF 工具
  - 创意: 属于你自己的文件王国
