# 50 Hacks

- **URL**: https://50hacks.co
- **一句话**: 互联网投票出来的 50 个最佳生产力技巧合集
- **详细说明**: 50 Hacks 是一个社区驱动的生产力窍门平台，用户无需注册即可提交技巧并为喜欢的内容投票。内容按热门、最新和最多票数排序，覆盖工作、学习、日常效率等各类实用小技巧。
- **目标用户**: 追求效率的上班族、学生、远程工作者
- **视觉风格**: 极简现代，卡片布局，橙色点缀
- **主色调**: 白色背景 + 橙珊瑚 (#EA5234) + 深色文字
- **氛围关键词**: 实用、社区感、轻量、激励、坦率
- **可归类方向**:
  - 常规: productivity / swiss-army-knife-sites
  - 创意: weekend-websites-worth-opening
