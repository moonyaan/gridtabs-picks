# Typora

- **URL**: https://typora.io
- **一句话**: 极简 Markdown 编辑器，所见即所得实时预览，写作和阅读合二为一
- **详细说明**: Typora 把 Markdown 的源码和预览合并为同一个视图——你输入的时候看到的就是最终效果，无需来回切换。支持数学公式、代码高亮（100+ 语言）、图表生成、表格编辑，界面干净，适合长期写作。
- **目标用户**: 作家、学生、开发者、内容创作者、喜欢 Markdown 的人
- **视觉风格**: 极简，干净，浅色主题为主，可自定义 CSS 主题
- **主色调**: 白色/浅色背景 + 红色 accent（#d14836）
- **氛围关键词**: 无干扰、专注、极简、自然书写体验、生产力
- **可归类方向**:
  - 常规: 写作工具、Markdown 编辑器、文档工具
  - 创意: 文字的禅，写作的回归
