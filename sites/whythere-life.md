# WhyThere

- **URL**: https://whythere.life
- **一句话**: 用真实数据对比全球城市，帮你决定下一站定居何处
- **详细说明**: WhyThere 提供城市横向对比工具，覆盖天气、生活成本、阳光时数、自然环境邻近度等维度，用可视化图表呈现。平台同时提供编辑故事内容，深度探讨城市间的住房可负担性、气候模式和生活方式差异，是数字游民和计划移居者的决策利器。
- **目标用户**: 数字游民、计划移居的专业人士、城市研究者
- **视觉风格**: 数据可视化为主，地图与图表结合，简洁现代
- **主色调**: 深蓝/紫色，配合数据图表色彩
- **氛围关键词**: 城市对比、数据驱动、移居决策、全球化、可视化
- **可归类方向**:
  - 常规: 生活工具 / 城市数据
  - 创意: 用数据找到你的下一个家
