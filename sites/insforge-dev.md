# InsForge

- **URL**: https://insforge.dev
- **一句话**: AI 编程代理的云基础设施平台，代理用 CLI 直接驱动数据库、存储和部署
- **详细说明**: InsForge 是专为 AI 编程代理设计的云基础设施，提供 Postgres 数据库、文件存储、边缘函数、应用部署和 AI 模型网关的完整套件。代理通过 CLI 和预置技能包端到端操控所有基础设施，开发者无需手动配置——代理发现、配置、部署，一气呵成。支持 Git 风格的后端"分支"功能。
- **目标用户**: AI 代理开发者、需要代理管理基础设施的工程师
- **视觉风格**: 开发者工具风格，简洁专业
- **主色调**: 深色 / 橙色强调
- **氛围关键词**: 代理原生、基础设施即代码、自动化、云、DevOps
- **可归类方向**:
  - 常规: keyboard-people
  - 创意: ai-no-code-2026
