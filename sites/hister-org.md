# Hister

- **URL**: https://hister.org
- **一句话**: 浏览记录全文搜索工具，本地索引，隐私优先
- **详细说明**: Hister 是"Web History on Steroids"，对访问过的网页进行全文内容索引，支持深度搜索。提供 Web UI 和 Terminal UI 两种界面，Chrome 和 Firefox 浏览器扩展。支持离线预览已索引网站、一键导入现有浏览器历史记录、搜索别名和高级查询语法。核心特色是隐私优先：本地索引浏览历史，全文搜索实际页面内容(而非仅 URL 和标题)，即时响应，无广告、无 AI 误导、无赞助排序、无结果操纵。开源 (AGPLv3 许可)。
- **目标用户**: 注重隐私保护的用户、需要频繁回溯浏览历史的效率型用户、厌恶广告/AI 错误建议和赞助结果的搜索用户、偏好键盘驱动工作流的技术人员
- **视觉风格**: 现代科技感，深色背景(rgb(10,10,18))，主文字为淡白色(#fafaff)，强调色使用蓝色(#3498db)、紫色(#8e44ad)、绿色(#2ecc71)渐变。渐变色标题动画、大圆角卡片(20px)、微妙的光晕和阴影效果、网格布局展示功能。
- **主色调**: 深色背景、蓝紫绿渐变
- **氛围关键词**: 隐私、极速、全文搜索、本地索引、开源
- **可归类方向**:
  - 常规：隐私工具、搜索工具、浏览器扩展、生产力工具、开源软件
  - 创意：记忆宫殿、本地优先、反广告、隐私革命、全文索引、键盘驱动
