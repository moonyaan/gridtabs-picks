# Bruno

- **URL**: https://www.usebruno.com
- **一句话**: 开源、离线优先、Git 原生的 API 客户端，Postman 的轻量替代品。
- **详细说明**: 将 API 请求集合以纯文本文件（.bru 格式）存储在本地文件系统，天然支持 Git 版本控制和团队协作。没有账号系统、没有云同步、完全离线运行，数据完全由用户掌控。支持 REST、GraphQL，内置脚本能力和 CI/CD 集成。
- **目标用户**: 后端开发者、全栈工程师、重视隐私和数据所有权的团队
- **视觉风格**: 品牌色为标志性的黄色/琥珀色，整体简洁克制，小狗 Logo 增添亲和力。极简路线，没有花哨动效，传达"工具就该好好做工具"的务实态度。
- **主色调**: 黄色/琥珀色、白色、深灰
- **氛围关键词**: 务实、独立、轻盈、开源精神、本地优先
- **可归类方向**:
  - 常规：开发工具、API 测试、开源软件、效率工具
  - 创意：车库里的黄色灯泡、独立精神、自由软件运动、工匠手感、小而美、反云端宣言
