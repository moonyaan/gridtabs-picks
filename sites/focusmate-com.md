# Focusmate

- **URL**: https://focusmate.com
- **一句话**: 虚拟共同工作平台，与陌生人配对开视频专注，用社交承诺对抗拖延
- **详细说明**: Focusmate 通过将你与一个陌生的"责任伙伴"配对进行 25/50/75 分钟的视频共同工作，利用"社会临场感"效应帮助你真正开始并完成任务。不需要聊天，只需开始工作，知道有人在旁边就足以克服拖延症。被 BBC、纽约时报、哈佛商评推荐，已有大量研究验证效果。
- **目标用户**: 拖延症患者、远程工作者、学生、需要专注力支持的自由职业者
- **视觉风格**: 简洁现代，紫蓝主色调，奶油白背景，亲切不压迫感
- **主色调**: 紫蓝 (#4648aa)、奶白 (#fcfaf7)、柔和绿蓝
- **氛围关键词**: 专注、社交责任、克服拖延、互助、虚拟共同工作
- **可归类方向**:
  - 常规: 生产力工具 / 专注辅助
  - 创意: 陌生人同行 / 社交能量驱动
