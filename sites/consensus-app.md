# Consensus

- **URL**: https://consensus.app
- **一句话**: AI 学术搜索引擎，2 亿篇论文中找答案
- **详细说明**: Consensus 是"AI for Research"学术文献搜索工具，搜索和分析 250M+ 同行评审文献。Deep Search 功能可自动扩展关键词、识别冲突观点、探索引用图谱，实现文献综述自动化。Medical mode 聚焦高质量医学资源(约 5 万临床指南和 800 万篇顶级医学期刊文章)。提供自然语言过滤器(时间范围、人群、研究设计等)和 Consensus Meter 可视化显示研究共识程度。
- **目标用户**: 研究人员(researchers)、学生(students)、临床医生(clinicians)、大学教职人员(faculty)。已有 1000 万研究人员、学生和临床医生使用
- **视觉风格**: 简洁专业的学术工具风格，强调清晰的信息层级和功能分区
- **主色调**: 专业学术风格
- **氛围关键词**: 学术、严谨、高效、权威、可视化
- **可归类方向**:
  - 常规：AI 工具、学术搜索、研究工具、教育工具、医学工具
  - 创意：知识加速器、共识可视化、文献综述革命、学术 AI、研究助手
