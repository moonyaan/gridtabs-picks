# Bun

- **URL**: https://bun.sh
- **一句话**: 极速的一体化 JavaScript 运行时和工具包
- **详细说明**: Bun 是新一代 JavaScript/TypeScript 运行时，比 Node.js 更快的执行速度。集成了运行时、包管理器、打包器、测试工具于一体，原生支持 TypeScript。兼容 Node.js API 和 Web API，为追求高性能的 Web 开发者提供一站式解决方案。
- **目标用户**: JavaScript/TypeScript 开发者、Node.js 用户寻求更快替代方案、前端框架开发者、追求高性能的 Web 开发者
- **视觉风格**: 现代科技感、极简、开发者友好。深色主题为主(#14151a)，粉色强调色(#f472b6)，紫色辅助色。可爱的小面包形象作为 Logo（有眼睛和腮红）。代码使用 JetBrains Mono 字体。
- **主色调**: 深灰背景(#14151a)、粉色强调(#f472b6)、蓝色(#00a6e1)
- **氛围关键词**: 极速、可爱、现代、开发者友好、性能至上
- **可归类方向**:
  - 常规：JavaScript工具、运行时、开发工具、性能优化、Node.js替代品
  - 创意：粉紫色系、可爱的技术、速度美学、面包意象、一体化哲学、性能革命
