# Anima

- **URL**: https://www.animaapp.com
- **一句话**: 把 Figma 文件、网址和自然语言提示转成生产就绪应用的 UX 设计代理
- **详细说明**: Anima 是 AI 驱动的设计到代码平台，可从 Figma 设计稿、截图或文字描述直接生成带数据库和部署的完整应用。支持品牌感知设计，让 AI 生成的界面忠实遵循你的设计系统。还支持 MCP 服务器集成，可嵌入其他 AI 代理工作流中。
- **目标用户**: 产品团队、设计师、开发者、使用 Figma 的前端工程师
- **视觉风格**: 现代极简技术界面，粗体排版，精致层次布局
- **主色调**: 珊瑚橙/三文鱼红 (#ff6250)，深灰背景，金黄和绿色点缀
- **氛围关键词**: 创新、智能、生产就绪、品牌感知、高效协作
- **可归类方向**:
  - 常规: 设计转代码 / AI 开发工具
  - 创意: Vibe Coding / 品牌智能设计
