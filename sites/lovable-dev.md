# Lovable

- **URL**: https://lovable.dev
- **一句话**: 面向非技术人员的 AI 全栈应用构建器，对话生成 + GitHub 同步 + 一键部署。
- **详细说明**: 定位为"超人级全栈工程师"，基于 TypeScript/React 技术栈，内置 40+ 组件库。比 Bolt 更重视设计质量和品牌调性，提供大量预制模板。GitHub 同步让开发者可以随时接手代码，不被平台锁定。
- **目标用户**: 独立创业者、没有开发经验的创始人、小企业主、产品设计师
- **视觉风格**: 暖色调极简设计，浅色背景带奶油色质感，毛玻璃效果，排版层次分明，留白充裕。像一本排版讲究的独立杂志。
- **主色调**: 奶油白/暖灰背景、深炭色文字、柔和品牌色点缀
- **氛围关键词**: 温暖、精致、可信赖、编辑感、手工质感
- **可归类方向**:
  - 常规：AI 工具、无代码平台、Web 开发、设计工具
  - 创意：阳光下的工作室、北欧极简、手作咖啡馆、温柔的力量
