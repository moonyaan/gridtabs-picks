# Emdash

- **URL**: https://emdash.sh
- **一句话**: 开源桌面应用，让你在同一代码库里并行运行多个 AI 编程代理（Claude Code、Codex、Cursor 等）
- **详细说明**: Emdash 是 YC W26 孵化的开源 Agentic 开发环境，每个代理在独立 Git worktree 中运行，互不干扰。内置 diff 对比、Issue 集成（Linear/GitHub/Jira）和 CI/CD 状态监控，支持 28+ 编程代理，代码和对话数据完全本地存储。
- **目标用户**: 软件工程师、使用 AI 编程代理的开发者
- **视觉风格**: 开发者工具风格，深色界面，技术感强，终端审美
- **主色调**: 深色/中性色调
- **氛围关键词**: 开源、并行代理、开发者工具、本地优先
- **可归类方向**:
  - 常规: AI 编程工具、开发者工具
  - 创意: keyboard-people、self-hosted-gems
