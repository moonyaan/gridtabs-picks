# Inclusive Colors

- **URL**: https://www.inclusivecolors.com
- **一句话**: 以无障碍为核心的调色板生成器，实时验证 WCAG/APCA 对比度，一键导出到 Tailwind、Figma、CSS
- **详细说明**: Inclusive Colors 使用 HSLuv 颜色模型（调整色相/饱和度不影响亮度），让设计师在创建色彩系统时就能确保无障碍合规。实时检查所有颜色组合的 WCAG 和 APCA 对比度，支持多色板创建，并可导出到 Tailwind、CSS 变量、Figma、Adobe Suite、Procreate 等主流设计工具，还内置真实 UI 预览效果。
- **目标用户**: UI/UX 设计师、前端开发者、设计系统团队、无障碍设计从业者
- **视觉风格**: 现代实用，以颜色本身为展示核心，专业而友好
- **主色调**: 随调色板变化，工具本身界面清晰中性
- **氛围关键词**: 无障碍设计、实用工具、设计系统、标准合规、色彩科学
- **可归类方向**:
  - 常规: 设计工具 / 色彩系统
  - 创意: 无障碍美学 / 包容性设计
