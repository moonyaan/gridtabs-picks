# OpenObserve

- **URL**: https://openobserve.ai
- **一句话**: 开源高性能可观测性平台，统一处理日志、指标和追踪，存储成本降低 140 倍
- **详细说明**: OpenObserve 是面向 AI 时代的开源可观测性平台，用 Rust 编写，采用高效列式存储和压缩技术，相比 Elasticsearch 可降低 140 倍存储成本。支持 PB 级规模数据处理，提供统一的日志、指标和链路追踪功能，被数千家企业（包括财富 500 强公司）所采用。
- **目标用户**: DevOps 和平台工程团队、寻求高性价比可观测性方案的企业和初创公司
- **视觉风格**: 现代极简设计，清晰布局，平滑动画，专业感强
- **主色调**: 蓝色 (#155dfc) 主色，白色/灰色中性色
- **氛围关键词**: 高效、高性能、可扩展、可靠、开源、透明、企业级
- **可归类方向**:
  - 常规: DevOps / 基础设施监控 / 开源工具
  - 创意: self-hosted-gems / keyboard-people
