# Cursor

- **URL**: https://cursor.sh
- **一句话**: AI 驱动的代码编辑器，让 AI 成为你的编程伙伴
- **详细说明**: Cursor 是专为 AI 时代设计的代码编辑器，集成了代码补全、AI Agent、代码库索引等功能。与 GitHub Copilot 等插件不同，Cursor 从底层架构就为 AI 协作优化，支持多模型切换（OpenAI/Anthropic/Gemini/xAI）。被超半数 Fortune 500 公司使用，客户包括 Stripe、OpenAI、NVIDIA、Figma 等。
- **目标用户**: 专业软件开发者、工程团队、企业开发团队
- **视觉风格**: 支持明暗双主题。浅色模式使用米白背景配深棕文字；深色模式使用深褐背景配浅灰文字。极简、现代、专业，大量留白，强调产品演示截图。
- **主色调**: 米白/深褐背景、深棕/浅灰文字
- **氛围关键词**: 专业、高效、AI原生、企业级、未来感
- **可归类方向**:
  - 常规：AI工具、代码编辑器、开发工具、企业软件
  - 创意：AI协作、双主题美学、专业主义、Fortune 500的选择、编程的未来
