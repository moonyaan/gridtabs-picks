# FasterGH

- **URL**: https://fastergh.com/
- **一句话**: 比 GitHub 官网快得多的 GitHub 镜像浏览体验，用 Convex 做实时缓存
- **详细说明**: FasterGH 是 GitHub 的快速镜像 UI，用 Convex 作为低延迟读取模型，页面加载时不直接命中 GitHub API，浏览大型仓库毫无卡顿。支持创建 Issue、评论、合并 PR 等写操作。开源项目，专为频繁在大型代码库间跳转的开发者设计。
- **目标用户**: 需要频繁浏览大型 GitHub 仓库的开发者、开源贡献者
- **视觉风格**: 与 GitHub 风格一致，偏向功能性和速度
- **主色调**: GitHub 风格中性色调
- **氛围关键词**: 快速、高效、开源、开发者工具、无卡顿
- **可归类方向**:
  - 常规: 开发者效率工具
  - 创意: GitHub 增强体验
