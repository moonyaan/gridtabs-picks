# Codelegate

- **URL**: https://codelegate.dev
- **一句话**: 键盘驱动的终端 UI，让多个 AI 编程代理在同一个代码库里并行工作。
- **详细说明**: Codelegate 利用 Git worktree 技术，让多个 Claude Code 或 Codex CLI 代理同时在同一仓库的不同分支工作，互不冲突。配备独立的终端、Git 管理界面，支持会话自动恢复、lazygit/tmux 集成，全键盘操作，专为 Mac/Linux 开发者设计。
- **目标用户**: 使用 AI 辅助编程（Claude Code、Codex CLI 等）的开发者，希望并行执行多个 AI 编码任务的工程团队
- **视觉风格**: 极简终端美学，TUI 界面，功能优先，无华丽图形
- **主色调**: 深色终端风格，单色或低调配色
- **氛围关键词**: 终端、开发者工具、AI编程、并行、键盘驱动
- **可归类方向**:
  - 常规: 开发者工具 / AI编程工具
  - 创意: AI原生开发工作流 / 构建基础设施
