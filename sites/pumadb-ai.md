# pumaDB

- **URL**: https://pumadb.ai
- **一句话**: AI 原生数据库，用自然语言替代 SQL 查询，让非技术用户也能分析数据
- **详细说明**: 将 AI 能力内置于数据库查询层，用户可以用自然语言提问来获取数据，不需要掌握 SQL 语法。降低了数据分析的技术门槛，让业务人员也能直接从数据库获取洞察。
- **目标用户**: 非技术业务人员、产品经理、数据分析师、创业公司全员
- **视觉风格**: 现代 AI 产品风格，简洁有力
- **主色调**: 深色/紫色或蓝色
- **氛围关键词**: AI、数据库、自然语言、无代码、分析
- **可归类方向**:
  - 常规: AI 工具、数据分析
  - 创意: 用说话代替写 SQL
