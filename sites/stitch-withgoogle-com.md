# Stitch by Google

- **URL**: https://stitch.withgoogle.com
- **一句话**: Google 出品的 AI UI 设计工具，上传草图或输入描述，自动生成生产可用的前端界面代码
- **详细说明**: Stitch 是 Google Labs 推出的实验性 AI 设计工具，利用 Gemini 多模态能力将自然语言描述或手绘草图直接转化为完整 UI 设计和前端代码。免费提供每月 350 次标准生成（Gemini 2.5 Flash）和 50 次高级生成（Gemini 2.5 Pro），支持导出到 Figma，极大压缩了从想法到界面的时间。
- **目标用户**: UI/UX 设计师、前端开发者、产品原型快速验证者
- **视觉风格**: 深色极简，Google 风格专业克制，深炭黑配白色字体
- **主色调**: 深炭色 (#191a1f)、白色、Google 品牌蓝
- **氛围关键词**: AI 设计、快速原型、Google Labs、实验性、未来工作流
- **可归类方向**:
  - 常规: AI 设计工具 / UI 生成
  - 创意: 草图变界面 / 设计加速器
