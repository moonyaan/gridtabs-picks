# Kilo Code

- **URL**: https://kilo.ai
- **一句话**: 开源 AI 编程代理 VS Code 扩展，多模型切换
- **详细说明**: Kilo Code 是 AI 驱动的编程代理，支持多平台(VS Code、JetBrains IDE、CLI)。提供代码生成和补全、自动化代码审查、云端代理执行、多平台部署支持、集成 500+ LLM 模型。强调开发者自主性：可自定义模型选择、完全透明操作(Apache-2.0 开源许可)、灵活定价、跨平台同步工作会话、企业级管理和安全。1.4M+ 活跃用户，20+ 万亿 token 已处理，OpenRouter 排名第一。
- **目标用户**: 寻求生产力提升的个人开发者、开发团队和企业、从竞品迁移的工程师(如 Cursor 用户)、使用 VS Code、JetBrains IDE 或 CLI 的工程师
- **视觉风格**: 深色主题，现代渐变强调。清晰极简界面设计。品牌色为青色/蓝绿色。JetBrains 字体族专业排版，高对比度提高可读性。
- **主色调**: 深色主题、青色/蓝绿渐变
- **氛围关键词**: 开源、高性能、多模型、开发者友好、透明
- **可归类方向**:
  - 常规：AI 工具、代码编辑器、开发工具、开源软件、IDE 扩展
  - 创意：开源精神、模型自由、代理编程、开发者赋能、多模型选择、反专有锁定
