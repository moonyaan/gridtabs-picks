# Mintlify

- **URL**: https://mintlify.com
- **一句话**: AI 原生开发者文档平台，代理自动监控代码库变更并提议更新文档，让文档和代码始终同步
- **详细说明**: Mintlify 将 Markdown 文件转化为精美文档网站，内置 AI 助手可对话式回答用户问题。独特的 Mintlify Agent 监控你的代码库，每次 ship 后自动提议文档更新，支持 llms.txt 和 MCP 标准，让 AI 工具也能高效读取你的文档。
- **目标用户**: 开发者、SaaS 公司、开源项目维护者、文档团队
- **视觉风格**: 专业简洁，强调美感和现代感，白色底色搭配清晰排版
- **主色调**: 白色/灰色系，亮色点缀
- **氛围关键词**: 文档、AI代理、开发者体验、自动更新
- **可归类方向**:
  - 常规: 文档工具、开发者工具
  - 创意: keyboard-people、swiss-army-knife-sites
