# Firma.dev

- **URL**: https://firma.dev
- **一句话**: 开发者签名基础设施，提供文档电子签名 API，方便 SaaS 内置合同签署功能
- **详细说明**: 面向开发者的电子签名 API 平台，让 SaaS 产品能够快速集成合同签署功能，无需自行搭建签名基础设施。提供简洁的 API 接口和 SDK，适合需要在产品内嵌入电子签名的开发团队。
- **目标用户**: SaaS 开发者、需要合同签署功能的产品团队、独立开发者
- **视觉风格**: 专业简洁，开发者工具风格
- **主色调**: 深色/白色，专业配色
- **氛围关键词**: 签名、API、开发者、合同、基础设施
- **可归类方向**:
  - 常规: 开发工具、电子签名
  - 创意: 合同签署的乐高积木
