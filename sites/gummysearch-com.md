# GummySearch

- **URL**: https://gummysearch.com
- **一句话**: Reddit 受众研究工具，从 13 万+ 子版块中挖掘用户痛点、需求和商业机会。
- **详细说明**: 专注 Reddit 社区数据深度挖掘，自动把帖子按痛点、求解决方案、花钱讨论、热门话题分类。支持关键词追踪和实时提醒。对独立开发者和 SaaS 创业者来说，这是"在用户说话的地方找答案"的工具。
- **目标用户**: 独立开发者、SaaS 创业者、内容创作者、营销人员
- **视觉风格**: 深色主题，搭配青绿色（teal）强调色。Dashboard 风格界面，数据可视化和筛选器为核心交互，注重信息密度和可读性。
- **主色调**: 深色底、青绿色（teal）
- **氛围关键词**: 数据驱动、洞察、社区、宝藏挖掘、Indie Hacker 气质
- **可归类方向**:
  - 常规：营销工具、用户研究、Reddit、数据分析、SaaS
  - 创意：淘金、地下社区、倾听者、独立精神、发现隐藏需求、创业者的雷达
