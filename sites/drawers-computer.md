# Drawers

- **URL**: https://drawers.computer
- **一句话**: macOS 应用，让每个项目拥有专属的 Dock、空间和窗口，彻底消除多项目切换时的干扰
- **详细说明**: Drawers 为 macOS 用户解决项目间上下文切换带来的分心问题，每个"抽屉"（项目）有自己的 Dock 和桌面空间，只显示当前项目所需的应用和资源，帮助维持心流状态。
- **目标用户**: 同时处理多个项目的 macOS 用户、自由职业者、开发者
- **视觉风格**: 极简主义，深色/中性色调，清晰无装饰，强调专注和秩序感
- **主色调**: 深色/中性色
- **氛围关键词**: macOS、专注、项目管理、心流状态
- **可归类方向**:
  - 常规: macOS 工具、生产力应用
  - 创意: focused-life、keyboard-people
