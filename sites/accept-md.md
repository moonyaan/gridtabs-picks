# accept-md

- **URL**: https://accept.md
- **一句话**: Next.js 中间件，一行命令让网页同时提供 Markdown 格式输出，对 LLM 友好。
- **详细说明**: 当客户端发送 Accept: text/markdown 请求头时，自动把 HTML 页面转换为干净的 Markdown 返回。支持 App Router、Pages Router、SSG、SSR、ISR，不需要改动任何现有代码。一条命令自动检测路由架构完成安装。MIT 开源。
- **目标用户**: Next.js 开发者、需要为 AI 爬虫暴露内容的团队、技术文档站点维护者
- **视觉风格**: 深色主题，蓝色调渐变品牌色。卡片式布局，代码块突出展示，CTA 按钮带发光效果。面向开发者的现代技术产品风格。
- **主色调**: 深黑、蓝色渐变
- **氛围关键词**: 开发者友好、优雅、极客、一行代码解决问题、克制
- **可归类方向**:
  - 常规：开发工具、Next.js、Markdown、AI/LLM、开源
  - 创意：暗夜蓝、代码之美、桥梁、极简主义、开发者的品味、未来基础设施
