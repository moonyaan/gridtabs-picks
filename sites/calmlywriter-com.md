# Calmly Writer

- **URL**: https://www.calmlywriter.com
- **一句话**: 极简无干扰写作工具，专注模式只高亮当前段落
- **详细说明**: Calmly Writer 是无干扰写作编辑器，专注模式(Focus mode)仅高亮当前编辑段落，输入时界面选项自动隐藏。提供简易格式化(选中文字点击样式或使用快捷键/Markdown)、自动保存与备份、字数统计与目标设定、多种颜色主题、导出为 Word 和 PDF 格式、打字机音效、智能标点、全屏模式。由独立开发者 Yusef Hassan 和 Amal Amrani 开发。
- **目标用户**: 需要专注写作、避免干扰的写作者（作家、内容创作者）
- **视觉风格**: 极简、低干扰，界面"simple, unobtrusive and ease-to-use"。提供浅色、深色、深蓝、沙丘色、复古等多种主题。
- **主色调**: 多主题可选
- **氛围关键词**: 专注、宁静、极简、无干扰、沉浸
- **可归类方向**:
  - 常规：写作工具、编辑器、生产力工具、专注工具
  - 创意：安静的书房、极简主义、专注模式、打字机美学、独立开发者温度、一次付费
