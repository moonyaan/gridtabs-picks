# Screen Studio

- **URL**: https://screen.studio
- **一句话**: macOS 录屏工具，自动缩放跟随，产品演示利器
- **详细说明**: Screen Studio 是专为 macOS 打造的专业录屏工具，以精致的视觉呈现与用户体验著称。核心特色是自动缩放跟随功能，让产品演示更专业。一次性购买定价模式(非订阅制)。
- **目标用户**: macOS 用户、内容创作者、教程制作者、需要高质量屏幕录制的专业人士
- **视觉风格**: 暗色主题，整体采用深色背景，营造专业、现代感。主色调为紫色(#4d2ff5)，背景色为纯黑(#000)与深灰(#13151b)，文字色为白色搭配不同透明度层级。圆角设计(6px-12px)，导航栏使用毛玻璃效果(backdrop-filter: blur(20px))，大量 0.3s 过渡动画，hover 状态响应快速(0.075s)，边框使用动态光线动画效果。
- **主色调**: 紫色(#4d2ff5)、纯黑(#000)、深灰(#13151b)、白色
- **氛围关键词**: 专业、精致、现代、流畅、产品级
- **可归类方向**:
  - 常规：macOS 应用、录屏工具、创作工具、演示工具
  - 创意：紫色系、自动魔法、产品演示美学、一次付费、专业主义、光效动画
