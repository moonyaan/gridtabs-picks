# Geo Racers

- **URL**: https://geo-racers.com
- **一句话**: 不坐飞机，从伦敦赛到东京——真实世界地理竞速游戏，有现金奖励
- **详细说明**: Geo Racers 是一个基于真实地理的竞速游戏，玩家在不乘飞机的情况下规划从一座城市到另一座城市的路线，比拼速度和地理知识。赛程如"Inverness 到 Gibraltar"，完赛有现金奖励。
- **目标用户**: 地理爱好者、休闲玩家、旅行控、竞速游戏玩家
- **视觉风格**: 地图中心的 Web 游戏界面，位置交互为核心
- **主色调**: 待确认（地图色系为主）
- **氛围关键词**: 冒险、探索、地理挑战、竞争、真实世界路线
- **可归类方向**:
  - 常规: 地理游戏、竞速、互动地图
  - 创意: 地球是游乐场、换种方式旅行
