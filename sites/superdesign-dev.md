# Superdesign

- **URL**: https://www.superdesign.dev
- **一句话**: 住在你 IDE 里的开源 AI 设计代理，一句话生成 10 个界面并行对比
- **详细说明**: Superdesign 是首个开源 AI 产品设计代理，直接集成到 Cursor、Windsurf、VS Code 中。用自然语言描述 UI，它能并行生成 10 个不同风格的设计方案，输出生产可用的 React 组件、Tailwind 样式和 CSS 动画。完全本地运行，MIT 开源，所有数据留在机器上。
- **目标用户**: 前端开发者、设计师、独立黑客、全栈工程师
- **视觉风格**: 现代技术前沿，深色主题为默认，Geist UI 风格
- **主色调**: 深色模式，黑白为主，系统偏好自适应
- **氛围关键词**: 开源、本地优先、并行创意、极速、设计赋能
- **可归类方向**:
  - 常规: AI 设计工具 / 开发者工具
  - 创意: Vibe Design / IDE 内设计工作室
