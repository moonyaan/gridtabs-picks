# Fueling Curiosity: The Great Refinery Run

- **URL**: https://fuelingcuriosity.com/game.html
- **一句话**: 交互式浏览器游戏，带你从采油到消费者完整走一遍石油精炼流程，STEM 教育利器。
- **详细说明**: 这是一款教育型网页游戏，玩家扮演炼油厂控制室操作员，依次完成脱盐、蒸馏、加氢处理、裂化等 15 个工艺单元，每个环节都有迷你互动任务。画面用 emoji 和彩色插图呈现，低门槛但内容扎实，适合家长、教育者和行业入门者。
- **目标用户**: 对化工/能源感兴趣的学生、家长、教师及行业从业新人
- **视觉风格**: 明亮卡通风，emoji 图标配色鲜艳，低压力的教育游戏美学
- **主色调**: 明亮多彩，黄色/橙色为主的工业色调
- **氛围关键词**: 教育游戏、STEM、互动、有趣、探索
- **可归类方向**:
  - 常规: STEM教育 / 互动学习
  - 创意: 互联网好奇心 / 无算法场所
