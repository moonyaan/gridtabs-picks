# Pinchflat

- **URL**: https://github.com/kieraneglin/pinchflat
- **一句话**: 自托管 YouTube 内容下载和媒体管理工具，让喜欢的频道内容永久保存在本地可离线查看
- **详细说明**: Pinchflat 是一个开源自托管应用，可以订阅 YouTube 频道或播放列表并自动下载新内容，将视频整理归档到本地媒体库。适合搭配 Plex、Jellyfin 等媒体服务器使用，彻底摆脱对 YouTube 在线访问的依赖。
- **目标用户**: 自托管爱好者、Home Lab 玩家、需要离线保存视频内容的用户
- **视觉风格**: 开发者项目风格，功能优先，简洁的 Web 管理界面
- **主色调**: 深色 + GitHub 绿
- **氛围关键词**: 自托管、YouTube、下载、媒体库、开源
- **可归类方向**:
  - 常规: 自托管工具 / 媒体管理
  - 创意: 把 YouTube 变成你自己的私人图书馆
