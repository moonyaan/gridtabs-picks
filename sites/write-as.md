# Write.as

- **URL**: https://write.as
- **一句话**: 隐私优先的极简博客平台，无广告无追踪，输入文字直接发布到互联网
- **详细说明**: Write.as 强调写作本身，提供匿名发布、自定义域名、多身份管理，并与 Mastodon 等去中心化网络联邦化。开源核心，付费版支持团队协作，是独立博主逃离算法平台的理想选择。
- **目标用户**: 注重隐私的个人博主、独立作家、希望脱离平台控制的创作者
- **视觉风格**: 极简黑白灰，干净排版，无任何视觉噪音，以内容为中心
- **主色调**: 黑色 + 白色 + 绿色强调 (#71D571)
- **氛围关键词**: 专注、独立、私密、极简、无干扰、开源
- **可归类方向**:
  - 常规: 写作工具、博客平台
  - 创意: 给文字一个没有算法的家
