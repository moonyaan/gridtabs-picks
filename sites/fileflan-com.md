# FileFlan

- **URL**: https://fileflan.com
- **一句话**: 无账号、无 Cookie、端对端加密的浏览器直连文件传输工具，文件从不经过服务器存储
- **详细说明**: FileFlan 实现真正的 P2P 文件传输——文件只在发送方和接收方浏览器之间流动，服务器不存储任何内容。支持最大 1GB 单文件、整个文件夹树发送，提供 CLI 模式供高级用户和自动化场景使用，完全免费且无需注册。
- **目标用户**: 隐私敏感用户、开发者、需要跨设备快速传文件的用户
- **视觉风格**: 极简主义，蓝色主色，强调简单和透明
- **主色调**: 蓝色
- **氛围关键词**: 隐私优先、P2P、无账号、加密传输
- **可归类方向**:
  - 常规: 文件传输工具、隐私工具
  - 创意: swiss-army-knife-sites、self-hosted-gems
