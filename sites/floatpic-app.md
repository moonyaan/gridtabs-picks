# FloatPic

- **URL**: https://floatpic.app
- **一句话**: macOS 极简无边框浮动图片查看器，原生手势+30+ 格式+OCR 色彩拾取
- **详细说明**: FloatPic 是一款 macOS 原生图片查看器，以"零干扰浮动窗口"为核心理念，没有标题栏没有边框，图片像漂浮在桌面上。支持捏合缩放/滑动翻页/双指旋转等原生手势，内置 EXIF 元数据（24+字段）、OCR 文字提取、AI 配色分析、像素颜色拾取、实时直方图和三模式图片对比，支持 100MB+ 大图的分块渲染。
- **目标用户**: 摄影师、设计师、macOS 重度用户
- **视觉风格**: 极简 macOS 原生，无边框浮动
- **主色调**: 系统级透明 / 白色
- **氛围关键词**: 极简、优雅、原生 macOS、无干扰、图片
- **可归类方向**:
  - 常规: make-it-beautiful、focused-life
  - 创意: pixel-and-handmade
