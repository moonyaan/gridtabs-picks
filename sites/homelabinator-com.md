# Homelabinator

- **URL**: https://homelabinator.com
- **一句话**: 基于 NixOS+K3s 的傻瓜式家庭服务器搭建平台，把自托管的门槛降到最低
- **详细说明**: Homelabinator 由一群学生开发者创建，目标是让自托管不再需要深厚的 Linux 知识——提供一个定制化的 NixOS 镜像，U 盘启动即可开始搭建 homelab 环境，内置 K3s 容器编排，适合想从零开始玩服务器的爱好者。
- **目标用户**: Homelab 爱好者、想尝试自托管但缺乏经验的技术入门者、开源社区用户
- **视觉风格**: 深色主题，极简技术感，社区开源氛围
- **主色调**: 深色背景 + 简洁排版
- **氛围关键词**: 开源、社区、自托管、入门友好、实验性
- **可归类方向**:
  - 常规: 自托管工具、开发者工具
  - 创意: 让每个人都能有自己的"云"
