# Prism.Tools

- **URL**: https://blgardner.github.io/prism.tools/
- **一句话**: 浏览器内运行的 38+ 开发者工具合集，零追踪、零广告、完全本地处理
- **详细说明**: Prism.Tools 是一个模块化的隐私优先开发者工具箱，包含 JSON 格式化、正则测试、Base64 编解码、SQL 格式化、JWT 解析、颜色/渐变生成、QR 码等 38 种以上常用工具。所有计算在浏览器本地完成，数据永不离开你的设备，完全免费开源。
- **目标用户**: 重视隐私的开发者、需要快速访问常用工具的前端/全栈工程师
- **视觉风格**: 现代暗色 glassmorphism 风格，紫粉渐变主色，卡片式布局，流畅微交互
- **主色调**: 深色背景 (#0a0a0f)，紫色 (#a78bfa) 到粉色 (#ec4899) 渐变
- **氛围关键词**: 隐私、高效、极简、开源、本地优先、开发者友好
- **可归类方向**:
  - 常规: 开发者工具 / 浏览器工具箱
  - 创意: keyboard-people / self-hosted-gems
