# Web Audio Studio

- **URL**: https://webaudio.studio
- **一句话**: 浏览器内 Web Audio API 可视化调试器，写代码同时实时看到音频节点图
- **详细说明**: Web Audio Studio 是专为开发者设计的音频编程工具，输入 Web Audio API 代码后立即生成交互式节点图，支持实时调节参数旋钮、在音频链任意节点插入信号检测。内置 20 个起手模板，从正弦波到 3D 空间音频一应俱全，完全基于浏览器，无需安装。
- **目标用户**: Web Audio API 开发者、音频程序员、DSP 学习者、对浏览器音频感兴趣的人
- **视觉风格**: 现代深色模式开发工具，极简 UI，等宽字体与无衬线字体搭配
- **主色调**: 深板岩蓝黑 (#11161b, #151a20)，琥珀黄强调 (#f59e0b)，数据可视化用霓虹绿和蓝
- **氛围关键词**: 技术、可视化、实时交互、教育性、精准
- **可归类方向**:
  - 常规: 开发者工具 / 音频编程工具
  - 创意: 音频可视化 / 声音可能性
