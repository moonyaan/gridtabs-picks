# GitClassic

- **URL**: https://gitclassic.com
- **一句话**: 无需臃肿 JS 的轻量 GitHub 浏览器，页面均在 14KB 以内
- **详细说明**: GitClassic 是一个极速轻量的 GitHub 瘦客户端，让开发者在没有 GitHub 官网那层繁重 JavaScript 的情况下浏览代码仓库。每个页面体积控制在 14KB 以内，页面加载极快，基于用户反馈持续迭代改进。适合在慢网络或追求极简体验的场景下浏览 GitHub。
- **目标用户**: 开发者，特别是在慢网络环境下工作、追求极简浏览体验或对 GitHub 臃肿感到烦恼的工程师
- **视觉风格**: 极简轻量，低 JS 渲染，以内容为主
- **主色调**: 默认 GitHub 风格的中性色调
- **氛围关键词**: 快速、轻量、极简、务实、无干扰
- **可归类方向**:
  - 常规: 开发者工具 / GitHub 客户端 / 效率工具
  - 创意: keyboard-people / swiss-army-knife-sites
