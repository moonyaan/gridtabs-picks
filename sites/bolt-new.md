# Bolt.new

- **URL**: https://bolt.new
- **一句话**: AI 驱动的全栈 Web 应用构建平台，通过对话生成完整的网站和应用。
- **详细说明**: 将多个 AI 编程代理整合到一个可视化界面，用户通过自然语言对话即可生成完整 Web 应用。内置云托管、数据库、用户认证、SEO 优化等基础设施。跟 Lovable 同属 AI 建站赛道，但 Bolt 更偏向全栈开发能力，强调"从想法到部署"的一站式体验。
- **目标用户**: 产品经理、创业者、营销人员、代理公司、学生
- **视觉风格**: 深色主题为主，搭配蓝紫色渐变光晕效果，赛博朋克气息。深色背景上叠加柔和的渐变发光，给人"正在运行中的终端"的感觉。
- **主色调**: 深灰/黑色背景、蓝紫色渐变光晕、白色文字
- **氛围关键词**: 未来感、极客、高效、赛博、夜间工作台
- **可归类方向**:
  - 常规：AI 工具、开发工具、无代码/低代码、Web 开发
  - 创意：深夜编码、霓虹灯下的车库、数字工厂、速度崇拜、黑客精神
