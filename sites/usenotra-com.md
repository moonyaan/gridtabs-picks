# Notra

- **URL**: https://www.usenotra.com
- **一句话**: 接入 GitHub、Linear、Slack，把你们团队每天 ship 的代码自动变成更新日志和博客草稿。
- **详细说明**: Notra 监控你的开发工作流，每次合并 PR 自动生成 changelog 条目，并根据品牌语气生成可发布的博客文章和社交媒体更新。学习团队的写作风格，一键接入，不需要手动整理发布内容。
- **目标用户**: 产品和工程团队、独立开发者、需要内容营销但不想费精力写更新日志的 SaaS 创始人
- **视觉风格**: 开发者友好的现代 SaaS 风，实用简洁，注重效率感
- **主色调**: 中性色调，专注内容本身
- **氛围关键词**: 自动化、开发者内容、工作流、效率、SaaS
- **可归类方向**:
  - 常规: 内容自动化 / 开发者工具
  - 创意: 构建中公开 / AI写作助手
