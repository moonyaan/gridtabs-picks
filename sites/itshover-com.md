# Its Hover

- **URL**: https://www.itshover.com
- **一句话**: 开源动态图标库，React/Next.js 交互动画 SVG
- **详细说明**: Its Hover 是开源的可编辑、动画 React 图标组件库，为现代界面和设计系统设计。每个图标都设计了动效作为功能特性而非装饰。完全可编辑的社区所有组件，可自定义。内置 shadcn 等组件库无缝集成。以实用性与美学精致性结合为特色，针对需要动画图标增强用户体验的开发者。
- **目标用户**: 使用 React、Next.js、shadcn 等组件库的开发者和设计师，需要动画图标的项目
- **视觉风格**: 现代、极简设计，强调动效和交互。支持明暗主题切换，基于现代网络技术(Next.js 框架)。
- **主色调**: 明暗双主题
- **氛围关键词**: 开源、动画、交互、意图设计、质量
- **可归类方向**:
  - 常规：开发工具、React 库、图标库、开源软件、设计系统
  - 创意：动效优先、SVG 美学、意图动画、开源精神、组件美学
