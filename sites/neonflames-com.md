# NeonFlames

- **URL**: https://neonflames.com
- **一句话**: 在纯黑画布上移动鼠标，轨迹化为迷幻霓虹火焰，人人都能创作出绝美光绘艺术
- **详细说明**: NeonFlames 是一个基于 Canvas 的在线绘画工具，鼠标（或手指）划过时会留下颜色艳丽、带有流动感的火焰光迹，无需任何绘画技巧即可生成视觉震撼的抽象艺术图。
- **目标用户**: 喜欢创作却无绘画基础的普通用户、数字艺术爱好者
- **视觉风格**: 纯黑背景，霓虹色系渐变光流，炫丽迷幻
- **主色调**: 黑色 + 霓虹多色（紫、橙、蓝、粉）
- **氛围关键词**: 霓虹、光绘、创作、迷幻、零门槛
- **可归类方向**:
  - 常规: 创意绘画工具
  - 创意: 不会画画也能做艺术家——鼠标即画笔，黑暗即画布
