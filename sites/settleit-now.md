# Settle It

- **URL**: https://settleit.now/
- **一句话**: 在群聊里发一个链接，秒速解决"去哪吃""看哪部电影"这类小决策
- **详细说明**: Settle It 让用户用自然语言输入选项（支持"或/逗号/斜线"等分隔符），一键生成可分享的投票链接，支持计时自动关闭，全程无需注册账号。是朋友群和小团队做快速决策的利器。
- **目标用户**: 日常决策困难的群组用户、小团队、任何需要快速问大家意见的场景
- **视觉风格**: 极简现代，亮色/暗色双模，圆角卡片，轻量感十足
- **主色调**: 白色/中性灰调 + 暗色模式下锌灰 (#3f3f46) 背景
- **氛围关键词**: 轻量、即时、社交、无摩擦、好玩、随意
- **可归类方向**:
  - 常规: productivity / decision-tools
  - 创意: weekend-websites-worth-opening / swiss-army-knife-sites
