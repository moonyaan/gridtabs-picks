# Knowza

- **URL**: https://knowza.ai
- **一句话**: AI 驱动的 AWS 认证备考平台，题库直接来自官方文档并配即时解析
- **详细说明**: Knowza 提供模拟真实 AWS 考试格式的计时练习题，AI 对每道错题给出详细解析。支持云从业者、解决方案架构师等 5 种认证，并有域级进度追踪和暗模式界面。
- **目标用户**: 备考 AWS 认证的开发者、云工程师、技术转型人员
- **视觉风格**: 干净现代，支持深色模式，AWS 橙色品牌点缀
- **主色调**: AWS 橙 (#FF9900) + 深色背景 + 绿色正确答案提示
- **氛围关键词**: 专业、支持性、学习导向、技术感、成就感
- **可归类方向**:
  - 常规: learning-tools / developer-tools
