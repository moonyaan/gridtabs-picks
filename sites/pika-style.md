# Pika

- **URL**: https://pika.style
- **一句话**: 截图美化工具，一键把截图变精美展示图
- **详细说明**: Pika 是截图美化与定制工具，可以为截图添加边框、背景、注释等。提供多种模板：App Store 截图、代码图片、推文截图、设备模型。支持 Chrome 扩展快速捕获网页、VSCode 扩展生成代码截图、URL 截图生成、社交媒体尺寸导出。被 YC 支持的初创公司及 Elgato、Incident.io 等企业使用。
- **目标用户**: 营销人员、自由网页设计师、开发者(代码截图需求)、社交媒体运营者、初创公司及企业用户
- **视觉风格**: 支持明暗主题切换，整体简洁现代。专业、简约，采用卡片式布局，强调视觉展示效果，以渐变背景和设备框架为特色。
- **主色调**: 简洁现代，明暗双主题
- **氛围关键词**: 美观、高效、专业、模板丰富、省时
- **可归类方向**:
  - 常规：设计工具、截图工具、营销工具、生产力工具
  - 创意：视觉美化、展示艺术、营销素材生成器、代码美学、社交媒体利器
