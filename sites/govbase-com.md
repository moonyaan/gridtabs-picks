# Govbase

- **URL**: https://govbase.com
- **一句话**: 从法案原文到媒体偏见到社交帖子，一站式追踪美国政策的全链路新闻平台
- **详细说明**: Govbase 直接从国会、白宫和联邦机构抓取法案和行政命令，用 AI 将复杂政策分解为通俗影响分析，同时展示左中右媒体的偏见评级报道，并关联政客在 X 和 Bluesky 上的社交发言，让你看清一项政策从起草到舆论的完整生命周期。
- **目标用户**: 关注美国政治的公民、记者、研究人员、政策分析师
- **视觉风格**: 信息密度适中，新闻应用风格，清晰层次结构
- **主色调**: 专业色系，白色为主，蓝色政治感调色板
- **氛围关键词**: 透明、中立呈现、信息驱动、政治参与、批判性
- **可归类方向**:
  - 常规: 政治资讯 / 公民科技工具
  - 创意: 算法去偏 / 互联网透明度
