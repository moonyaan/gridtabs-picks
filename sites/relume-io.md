# Relume

- **URL**: https://relume.io
- **一句话**: AI 驱动的网站设计平台，分钟级生成站点地图和线框图
- **详细说明**: 为设计师、开发者和代理商打造的 AI 设计工具。快速生成营销网站的站点地图、线框图和样式指南，无缝对接 Webflow、Figma 和 React 工作流。
- **目标用户**: 使用 Webflow、Figma 和 React 的设计师、开发者、设计代理商和自由职业者
- **视觉风格**: 现代极简风，配合有趣的互动元素、动画光标和角色插画
- **主色调**: 紫色主色 (#6248FF) + 白色背景 + 橙红强调色 + 渐变动画
- **氛围关键词**: 专业友好、高效、创意赋能、创新、协作、有趣不失严肃
- **可归类方向**:
  - 常规: 设计工具、AI 工具、网站构建、协作软件
  - 创意: AI 设计师助手、分钟级创意实现、从想法到线框的桥梁
