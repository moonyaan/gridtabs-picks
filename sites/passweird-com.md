# Passweird

- **URL**: https://www.passweird.com
- **一句话**: 生成奇葩但好记的密码，把安全要求变成一门荒诞艺术
- **详细说明**: Passweird 生成符合大小写、数字、特殊符号等安全要求的密码，但组合方式故意搞怪，让人忍不住发笑的同时也更容易记住。把密码管理这件枯燥的事变得有趣。
- **目标用户**: 需要创建强密码但又记不住的普通用户
- **视觉风格**: 轻松幽默，色彩明亮，带插画或趣味排版
- **主色调**: 鲜艳多彩
- **氛围关键词**: 趣味、密码、安全、幽默、记忆
- **可归类方向**:
  - 常规: 安全工具 / 密码生成器
  - 创意: 安全不必无聊——让你的密码成为一句荒诞的格言
