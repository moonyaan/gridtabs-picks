# Databox MCP

- **URL**: https://databox.com/mcp
- **一句话**: MCP 服务器让你在 Claude、ChatGPT 或 Cursor 里直接用自然语言查询 Databox 业务数据。
- **详细说明**: Databox MCP 是一个模型上下文协议服务器，将 Databox 的业务数据仪表盘接入 Claude、ChatGPT、n8n 等 AI 工具，使用户能用自然语言提问并获得准确的数据洞察。与直接上传数据到 AI 不同，MCP 查询的是真实 Databox 数据，结果与仪表盘一致、不产生幻觉。Databox 用户免费使用，无需额外订阅。
- **目标用户**: 使用 Databox 的营销和业务团队、需要在 AI 工具里访问业务数据的分析师
- **视觉风格**: 专业企业级界面，数据可视化图表为主，蓝色品牌色系
- **主色调**: 品牌蓝、白色背景、中性灰
- **氛围关键词**: 数据驱动、AI集成、MCP、业务分析、无缝
- **可归类方向**:
  - 常规: 数据分析/AI集成工具
  - 创意: 让你的 AI 助手真正读懂你的业务数字
