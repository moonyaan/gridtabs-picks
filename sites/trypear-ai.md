# PearAI

- **URL**: https://trypear.ai
- **一句话**: 开源 AI 代码编辑器，基于 VSCode，YC 支持
- **详细说明**: PearAI 是"The AI Code Editor For Your Next Project"，提供 PearAI Router(自动连接最优 AI 模型，单一订阅获取市场最佳模型)、PearAI Agent(基于 Roo Code/Cline 的编程代理，自动编写功能代码和修复 bug)。即将推出 PearAI Creator(从零创建新项目)、PearAI Login(用户登录功能实现)、PearAI Launch(Web 项目发布)。定位为"made for makers of any technical level"。
- **目标用户**: 各技术水平的创造者，包括曾使用 ChatGPT/Claude 辅助编程的开发者、学习新编程语言的开发者、做副业项目的工程师
- **视觉风格**: 极简现代风格，以白色(#FFFFFF)和浅灰(#FBFBFB, #F4F4F4)为主背景，亮绿色(#B3F353)作为强调色。大量使用圆角元素(rounded-xl)，支持明暗主题切换。清晰的卡片式布局，充足留白。
- **主色调**: 白色/浅灰背景、亮绿色(#B3F353)强调
- **氛围关键词**: 开源、易用、Y Combinator、模型优势、成本节约
- **可归类方向**:
  - 常规：AI 工具、代码编辑器、开发工具、开源软件、YC 项目
  - 创意：绿色系、开源精神、成本节约、YC 背书、模型路由器、副业利器
