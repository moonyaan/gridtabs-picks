# DynamicLake

- **URL**: https://www.dynamiclake.com
- **一句话**: 把 iPhone 动态岛体验带到 Mac 的系统增强应用
- **详细说明**: DynamicLake 是一款专为 Mac 设计的系统工具，将 iPhone Dynamic Island 的动态通知和信息展示体验移植到 MacBook。支持音乐控制、天气、日历、FaceTime 通知、iMessage/WhatsApp/Telegram/Slack 消息即时展示，并可拖放操作。2026 年重大更新引入完整 Slack 通知系统和多文件转换支持。
- **目标用户**: Mac 用户，特别是追求系统增强功能和现代 UI 体验的 MacBook 用户
- **视觉风格**: 现代极简风格，圆角设计元素，浅灰背景配深色文字
- **主色调**: 浅灰 (#ededed)，白色，深灰/黑色，浅蓝链接 (#09f)
- **氛围关键词**: 动态、创新、简洁、流畅、科技感
- **可归类方向**:
  - 常规: macOS 系统工具 / 系统增强 / 生产力工具
  - 创意: focused-life / keyboard-people
