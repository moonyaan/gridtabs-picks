# Pointer Pointer

- **URL**: https://pointerpointer.com/
- **一句话**: 把鼠标停在任意位置，自动找到一张有人指着该位置的照片
- **详细说明**: 一个荒诞又精妙的互动艺术作品，无论你把鼠标移到屏幕哪里，网站都能找到一张照片，照片中有人的手指正好指向你光标所在的位置。
- **目标用户**: 所有人，追求奇妙体验
- **视觉风格**: 极简，以图片为主
- **主色调**: 白色背景
- **氛围关键词**: 荒诞、艺术、互动、奇妙、人类
- **可归类方向**:
  - 常规: 互动艺术、趣味网站
  - 创意: 互联网奇景
