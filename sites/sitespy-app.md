# Site Spy

- **URL**: https://sitespy.app
- **一句话**: 监控任意网页变化，把每个页面的改动变成 RSS 订阅或即时通知。
- **详细说明**: Site Spy 按你设定的间隔抓取网页快照，用绿色/红色高亮对比变化，支持精确到页面元素级别的追踪。可导出 RSS 订阅，集成 MCP 协议让 AI 助手也能读取变动数据，还支持 Telegram 告警。
- **目标用户**: 开发者、数据分析师、价格追踪者、需要实时感知网页变化的信息工作者
- **视觉风格**: 现代 SaaS 风格，深色/浅色主题切换，浏览器 Mockup 配图，卡片式布局
- **主色调**: 青色（Starter）、紫色（Pro）、中性灰底
- **氛围关键词**: 专业、实用、监控感、开发者友好、信息流
- **可归类方向**:
  - 常规: 网页监控 / 开发者工具
  - 创意: 信息流控制 / always-on互联网
