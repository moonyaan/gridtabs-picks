# Formula Bot

- **URL**: https://formulabot.com
- **一句话**: 上传表格用自然语言提问，AI 自动生成 Excel 公式、图表和数据洞察
- **详细说明**: Formula Bot 支持 Excel、Google Sheets、Snowflake 和多种 SQL 数据库，用户通过对话方式获取数据分析结果、公式生成和可视化报表。数据处理在隔离沙箱环境中完成，保障隐私安全。
- **目标用户**: 业务分析师、非技术背景的运营和销售人员、需要快速获取数据洞察的团队
- **视觉风格**: 现代简洁，温暖珊瑚色调，专业又亲切
- **主色调**: 暖珊瑚/三文鱼色 (#da7756) + 米白背景 + 深炭黑
- **氛围关键词**: 高效、无门槛、智能、可信、专业
- **可归类方向**:
  - 常规: ai-tools / productivity
  - 创意: swiss-army-knife-sites
