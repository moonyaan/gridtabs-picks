# Respectify

- **URL**: https://respectify.org
- **一句话**: AI 评论审核工具，不是简单屏蔽而是教用户怎么更好地表达——边发言边成长
- **详细说明**: Respectify 在评论发出前用 AI 分析语气、逻辑和尊重程度，给用户即时反馈并引导修改，而不是单纯删除或封禁。支持 WordPress 集成和 API 接入，帮助社区主人在保护讨论质量的同时教化用户。
- **目标用户**: 社区站长、论坛运营者、博客主、网站管理员
- **视觉风格**: 现代极简，磁力按钮动效，深色模式支持
- **主色调**: 紫色 (#8b5cf6) + 蓝绿彩虹渐变
- **氛围关键词**: 建设性、包容、教化、社区、AI 调解
- **可归类方向**:
  - 常规: 社区管理工具、内容审核、AI 工具
  - 创意: 让网络更文明、惩罚之外的可能性
