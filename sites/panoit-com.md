# Pano

- **URL**: https://www.panoit.com
- **一句话**: 保存任意网址并组织到可分享书架的个人网络图书馆
- **详细说明**: Pano 是一个网络内容收藏平台，允许用户保存任何 URL 并组织到"书架"中，支持颜色编码收藏集、标签和搜索。用户可分享个人书架或关注他人的策展品味，构建互联网上的个人图书馆。跨浏览器使用，提供免费服务。
- **目标用户**: 知识工作者、内容策展人、研究人员、对发现和组织互联网资源感兴趣的创意专业人士
- **视觉风格**: 极简主义现代设计，干净排版，品牌指定色彩系统
- **主色调**: 中性色调（白色背景，深色文本），强调色用于交互元素
- **氛围关键词**: 策展、秩序、发现、品味、个人收藏、共享知识
- **可归类方向**:
  - 常规: 书签管理 / 内容策展 / 知识组织
  - 创意: focused-life / weekend-websites-worth-opening
