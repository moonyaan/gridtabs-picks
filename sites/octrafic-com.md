# Octrafic

- **URL**: https://octrafic.com
- **一句话**: 用自然语言从终端测试 API，开源 AI CLI，无需写脚本无需配置文件
- **详细说明**: Octrafic 是一个开源 CLI 工具，让你用纯英文描述想测试什么，AI 自动生成请求、执行测试并校验响应，支持导出为 Postman Collection、pytest 文件或 PDF 报告。单个 Go 二进制文件，支持 OpenAI/Anthropic/Ollama 等多个 LLM 提供商，完美集成 CI/CD 流程。
- **目标用户**: 后端开发者、DevOps 工程师、QA 专业人士、键盘党工作流爱好者
- **视觉风格**: 暗色终端美学，霓虹青色强调，等宽字体界面，ASCII 艺术品牌标识
- **主色调**: 深色背景 (#141414)、青色 (#38bdf8)、绿色成功态 (#34d399)、红色错误态 (#fb7185)
- **氛围关键词**: terminal、open-source、developer-tool、AI-native、lightweight
- **可归类方向**:
  - 常规: 开发者工具、API 测试、CLI 工具
  - 创意: keyboard-people、swiss-army-knife-sites
