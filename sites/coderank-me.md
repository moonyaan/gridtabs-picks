# Coderank

- **URL**: https://coderank.me
- **一句话**: 把 GitHub 公开活动转化为开发者排名和项目趋势的智能分析平台。
- **详细说明**: Coderank 解析开发者的贡献深度、一致性、势头和项目吸引力，生成可比较的 CodeRank / Tastemaker / Influence 分数。支持自然语言 AI 搜索找开发者和项目，覆盖语言、地点、公司等多维度筛选。
- **目标用户**: 工程团队 leader、开源研究者、想找协作者的独立开发者、招聘工程师的 HR
- **视觉风格**: 现代简洁科技风，支持深/浅主题，数据驱动的卡片和排行榜布局
- **主色调**: 中性灰/白为主，配色低调专业
- **氛围关键词**: 数据驱动、开发者文化、排行榜、专业、发现感
- **可归类方向**:
  - 常规: 开发者工具 / 开源发现
  - 创意: 互联网好奇心 / 排行榜文化
