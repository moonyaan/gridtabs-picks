# The Useless Web

- **URL**: https://theuselessweb.com
- **一句话**: 一键随机传送到互联网上最没用但最好玩的奇葩网站，每次都是惊喜
- **详细说明**: 点击页面上的按钮，会被随机导向一个毫无实际用途却极度有趣的网站。收录了数百个互联网奇葩角落，是探索网络荒诞文化的最佳随机入口，每次体验都不一样。
- **目标用户**: 无聊想消遣的人、互联网文化爱好者、喜欢随机探索的人
- **视觉风格**: 简单醒目，大按钮设计，带一点故意的土味美学
- **主色调**: 白色 + 醒目按钮色
- **氛围关键词**: 随机、无聊、荒诞、探索、互联网文化
- **可归类方向**:
  - 常规: 娱乐导航 / 随机网站推荐
  - 创意: 互联网的盲盒——每次点击都是一次对人类创造力下限的探索
