# Crawler.sh

- **URL**: https://crawler.sh
- **一句话**: 本地优先的网站爬虫，几秒内完成爬取、SEO 分析和内容导出
- **详细说明**: Crawler.sh 是一个用 Rust 构建的高速本地 Web 爬虫，支持 CLI 和桌面应用。可以爬取整站、自动执行 22 项 SEO 检查、将页面内容提取为干净的 Markdown，并导出为 JSON、Sitemap XML、CSV 等格式。所有处理均在本地完成，无需上传，数据完全私密。
- **目标用户**: SEO 专业人员、内容团队、开发者、技术运营人员
- **视觉风格**: 现代极简，代码优先美学，深色模式为主
- **主色调**: 深海军蓝/碳灰，白色与浅灰色调
- **氛围关键词**: 高速、本地优先、隐私、开发者友好、精准
- **可归类方向**:
  - 常规: SEO 工具 / 开发者工具
  - 创意: 极客必备 / 极速本地工具
