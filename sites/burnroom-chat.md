# Burn Room

- **URL**: https://burnroom.chat
- **一句话**: 消息1小时后自动销毁的临时 SSH 聊天室，无需安装任何客户端
- **详细说明**: Burn Room 是一个通过浏览器直接访问的 SSH 终端聊天服务，所有消息在1小时后彻底消失。无需注册账号，打开即聊，关闭即焚。采用 WebSocket 实现实时双向通信，支持移动端触控操作，适合需要一次性安全对话的场景。
- **目标用户**: 开发者、安全从业者、需要临时私密沟通的技术人群
- **视觉风格**: 极简终端美学，深黑色背景配橙红色品牌色，monospace 字体，复古赛博感
- **主色调**: 深黑 #0a0a0a、品牌橙红 #ff4500、连接绿 #22c55e
- **氛围关键词**: 临时、极简、私密、终端感、赛博朋克、低摩擦
- **可归类方向**:
  - 常规: 隐私工具 / 开发者工具
  - 创意: 数字焚书、消失的对话
