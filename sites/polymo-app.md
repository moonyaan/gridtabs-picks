# Polymo

- **URL**: https://www.polymo.app
- **一句话**: 用 AI 构建单页 HTML Web 应用的极简工具，像拍证件照一样快速出原型
- **详细说明**: Polymo 提供稳定的单页 HTML 应用脚手架，集成 TypeScript、React、Tailwind CSS 等现代技术栈。专注于让想法快速成形，在用其他更复杂工具投入前先验证概念。界面极简，上手零门槛。
- **目标用户**: 独立开发者、快速原型验证者、想试想法的创业者
- **视觉风格**: 极简主义，像素字体 (GeistPixel)，网格几何感强
- **主色调**: 深蓝灰 (#25343F)，白色，柔和辅助色
- **氛围关键词**: 直接、高效、实用、零门槛、原型友好
- **可归类方向**:
  - 常规: AI 应用构建 / 快速原型工具
  - 创意: 极简开发 / 想法孵化器
