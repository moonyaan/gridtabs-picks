# Paperboat

- **URL**: https://paperboat.website/home/
- **一句话**: 无广告、无 JS、无噪音的个人博客和作品站平台，把想法折成纸船送给世界
- **详细说明**: Paperboat 是一个极简的个人网站和博客托管平台，无需 JavaScript，无广告，内置 RSS，支持 Markdown 和富文本写作。强调社区与 webring 连接，月费 €5 可解锁高级功能，设计哲学是"轻量、友善、可及"。
- **目标用户**: 独立博主、个人作品展示者、想拥有一块安静网络角落的人
- **视觉风格**: 极简，无 JS，自然系美学，轻盈而个人化
- **主色调**: 自然系（具体颜色未明确）
- **氛围关键词**: 轻盈、友善、无噪音、个人、社区、慢互联网
- **可归类方向**:
  - 常规: 博客平台、个人网站、写作发布
  - 创意: 慢互联网、纸船哲学、私人角落
