# DEV Community

- **URL**: https://dev.to
- **一句话**: 开发者分享和交流的社区
- **详细说明**: DEV Community 是开发者知识分享平台，帮助开发者保持行业趋势更新并推进职业发展。提供文章、挑战赛、教育课程、视频和社区讨论。基于 Forem 开源框架构建，支持标签式内容组织、用户资料、反应系统(点赞、独角兽、火焰图标)。3728129 个开发者社区，涵盖网页开发、AI、DevOps、安全等。核心卖点是免费包容社区、广泛标签系统、有形回报的挑战赛(GitHub Universe 门票、Copilot 订阅)、开源自定义、教育课程、活跃参与功能。
- **目标用户**: 开发者社区，从初学者到有经验的专业人士
- **视觉风格**: 现代清晰界面，开发者友好美学，标签式组织，反应系统，响应式导航
- **主色调**: 现代开发社区风格
- **氛围关键词**: 社区、知识分享、开发者、学习、包容
- **可归类方向**:
  - 常规：开发社区、学习平台、知识分享
  - 创意：开发者聚集地、开源精神、社区驱动
