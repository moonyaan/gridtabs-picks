# Monica

- **URL**: https://www.monicahq.com
- **一句话**: 开源个人 CRM，帮你记录和维护与亲友之间的每一段重要关系
- **详细说明**: Monica 是一款开源的个人关系管理工具，让用户记录联系人信息、互动历史、重要纪念日和提醒，帮助忙碌的人维系真正重要的人际关系。强调隐私与数据所有权，支持自建部署。界面清晰，操作简单。
- **目标用户**: 注重人际关系但容易遗忘细节的忙碌专业人士、注重隐私的技术用户
- **视觉风格**: 简洁明亮，专业中带温情
- **主色调**: 蓝白为主
- **氛围关键词**: 人情味、隐私、开源、关系管理、温暖
- **可归类方向**:
  - 常规: 个人 CRM / 生产力工具
  - 创意: 数字时代的人情味 / 自托管精品
