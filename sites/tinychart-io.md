# TinyChart

- **URL**: https://tinychart.io/
- **一句话**: 粘贴 CSV 数据，一秒生成可分享的图表链接，无需注册
- **详细说明**: TinyChart 极度简单：粘贴数据或上传 CSV，选择图表类型，立刻获得可分享的链接。没有账号注册，没有复杂配置，专为快速可视化而生。适合临时展示数据、发给别人看一眼，而不是做复杂的 BI 分析。
- **目标用户**: 数据分析师、教师、需要快速制图分享的任何人
- **视觉风格**: 极简，功能优先，无多余装饰
- **主色调**: 干净中性配色
- **氛围关键词**: 高效、无摩擦、轻量、实用、即时
- **可归类方向**:
  - 常规: 数据可视化工具
  - 创意: 瞬间分享的轻工具
