# Is It a Pint?

- **URL**: https://isitapint.com
- **一句话**: 一场草根调查运动，用数据揭露酒吧是否真的给你倒了满满一品脱
- **详细说明**: Is It a Pint? 是一个消费者权益倡导平台，收集全球酒吧和餐厅的品脱饮料实际容量数据，发起"品脱巡逻"运动。网站呈现调查方法、数据统计和讨论，揭示消费者在饮料分量上被欺骗的现象，带有幽默感和社区精神。
- **目标用户**: 酒吧爱好者、消费者权益关注者、对数据驱动调查感兴趣的网友
- **视觉风格**: 深海军蓝背景，金黄色强调，现代无衬线字体，带光晕动效的戏剧感
- **主色调**: 深蓝 (#0f0f1e) + 金黄 (#f5c842)
- **氛围关键词**: 草根、幽默、数据驱动、消费者权益、社区调查
- **可归类方向**:
  - 常规: 消费者工具、社区数据
  - 创意: 互联网的品脱侦探
