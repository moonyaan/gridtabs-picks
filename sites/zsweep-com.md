# Zsweep

- **URL**: https://zsweep.com
- **一句话**: 用 Vim 快捷键操控的网页版扫雷，键盘党的新欢
- **详细说明**: Zsweep 是扫雷游戏的现代网页实现，特色是支持 Vim 风格的键盘操控（hjkl 移动、空格旗帜/和弦），提供 9x9、16x16、30x16 三种难度，还有主题切换和游戏数据统计。对于习惯键盘的开发者而言，这是最顺手的扫雷版本。
- **目标用户**: 开发者、键盘流用户、扫雷爱好者
- **视觉风格**: 简洁复古，命令行/终端美学
- **主色调**: 深色终端风，配色简洁
- **氛围关键词**: 扫雷、Vim、键盘操控、复古游戏、开发者文化
- **可归类方向**:
  - 常规: 游戏 / 开发者娱乐
  - 创意: 给 Vim 党的扫雷
