# The Wiki Game

- **URL**: https://www.thewikigame.com
- **一句话**: 从一个维基百科页面出发，只能点内链，用最少步骤抵达目标页面
- **详细说明**: 游戏给出起点和终点两个维基百科词条，玩家只能通过点击页面内的超链接跳转，挑战用最少的点击次数从起点走到终点。考验联想能力和对知识网络结构的理解，也能让人在不知不觉中学到大量冷知识。
- **目标用户**: 维基百科爱好者、知识游戏玩家、好奇心重的人
- **视觉风格**: 维基百科原生界面风，简洁朴素，带游戏化进度提示
- **主色调**: 白色 + 蓝色链接色
- **氛围关键词**: 知识、跳转、联想、维基、益智
- **可归类方向**:
  - 常规: 知识游戏 / 益智娱乐
  - 创意: 知识的六度分隔——用超链接穿越整个人类百科全书
