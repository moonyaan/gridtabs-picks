# ContentIdeas

- **URL**: https://contentideas.io
- **一句话**: 免费发现任意话题下传播最广的内容，按社交分享量排序，无需注册
- **详细说明**: ContentIdeas.io 是一个内容发现工具，输入关键词、域名或作者名，立即获取该话题下社交分享量最高的文章和内容。支持按平台、语言和多关键词组合过滤，既可用于发现竞品内容策略，也能为自己的博客和社媒找灵感。完全免费，背后有自建爬虫和搜索引擎。
- **目标用户**: 内容营销人、社媒运营、博主、SEO 从业者、独立创作者
- **视觉风格**: 现代简洁，绿色系主色调，Mantine UI 框架，专业感
- **主色调**: 深绿到浅绿渐变、白色背景、蓝紫辅助色
- **氛围关键词**: 免费、数据驱动、内容研究、社交验证、无需注册、高效
- **可归类方向**:
  - 常规: 内容工具 / 市场研究
