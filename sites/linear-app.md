# Linear

- **URL**: https://linear.app
- **一句话**: 为现代软件团队打造的极简高效项目管理工具
- **详细说明**: Linear 专为工程团队设计的问题追踪和项目管理工具，以极致的速度和流畅的键盘操作体验著称。与 Jira 等传统工具不同，Linear 抛弃了复杂的配置和臃肿的界面，专注于让团队快速创建、分类和追踪任务。支持 AI 辅助开发、Sprint 周期管理、100+ 工具集成。界面反应速度快到几乎感觉不到延迟。
- **目标用户**: 现代软件产品团队、工程师、产品经理、设计师、从初创公司到大型企业
- **视觉风格**: 极简主义、深色模式优先、大量留白、精致的微交互动画、注重排版层级。深色主题(#08090a)，使用 Inter Variable 字体，强调可读性。整体感受专业、精致、科技感强。
- **主色调**: 深灰/黑色背景(#08090a)、紫色/蓝紫色高亮、白色文字、低饱和度的状态色
- **氛围关键词**: 快速、专业、克制、未来感、工具美学
- **可归类方向**:
  - 常规：项目管理、团队协作、开发者工具、任务追踪、敏捷开发、AI工具
  - 创意：极简美学、速度至上、键盘优先、深色系工具、工程师文化、去繁就简、专注力工具、反臃肿设计
