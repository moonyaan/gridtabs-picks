# TinyCld

- **URL**: https://tinycld.org
- **一句话**: 自托管生产力套件，提供邮件、日历、联系人、云存储六合一，永久免费
- **详细说明**: TinyCld 是一个开源的自托管生产力平台，整合了邮件、日历、联系人、文档和云存储功能，支持 IMAP、CalDAV 等标准协议，可与其他客户端完全兼容。完全本地部署，数据不经过第三方服务器，提供移动 App（iOS/Android）配合使用。
- **目标用户**: 重视数据隐私的团队、开发者、想要脱离订阅制 SaaS 的小企业用户
- **视觉风格**: 现代极简主义，清晰排版层级，充足留白，响应式布局
- **主色调**: 珊瑚红/橙红色（#EB7D5E）搭配深色背景，蓝色强调色
- **氛围关键词**: 自由、隐私、自托管、开源、掌控
- **可归类方向**:
  - 常规: self-hosted-gems, swiss-army-knife-sites
  - 创意: solo-founder-back-office
