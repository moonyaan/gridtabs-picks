# Web Speed

- **URL**: https://www.getwebspeed.io
- **一句话**: AI 代理共享的开放 Web 地图注册表，让代理知道如何导航互联网
- **详细说明**: Web Speed 是一个开源 MCP 工具和共享注册表，为 AI 代理提供网络导航的"地图"——告诉代理哪些网站有什么内容、如何交互。解决了 AI 代理在浏览互联网时反复"重新发现"相同站点结构的问题，通过共享知识大幅提升代理效率。
- **目标用户**: AI 代理开发者、MCP 生态系统构建者
- **视觉风格**: 开发者工具风格，简洁
- **主色调**: 白色 / 深色
- **氛围关键词**: AI代理、MCP、开源、互联网基础设施、协作
- **可归类方向**:
  - 常规: keyboard-people
  - 创意: ai-on-trial
