# Neal.fun Internet Artifacts

- **URL**: https://neal.fun/internet-artifacts
- **一句话**: 互动式早期互联网博物馆
- **详细说明**: Neal.fun 的 Internet Artifacts 是一个精心收集的早期互联网经典时刻的互动式博物馆。包含 57 个代表性的网络文化、应用和现象，通过交互式体验让用户重温互联网发展历史。
- **目标用户**: 互联网文化爱好者、数字历史研究者、怀旧爱好者、年轻一代发现历史
- **视觉风格**: 复古互联网风格，创意十足，互动式设计
- **主色调**: 90/00 年代互联网配色
- **氛围关键词**: 怀旧、互联网历史、互动、文化、经典
- **可归类方向**:
  - 常规: 互联网博物馆、文化档案、历史参考
  - 创意: 互联网考古、怀旧体验、文化时间囊
