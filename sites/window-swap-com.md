# WindowSwap

- **URL**: https://window-swap.com
- **一句话**: 打开陌生人家的窗户，看看世界各地的真实窗外风景，互联网上最平静的地方
- **详细说明**: WindowSwap 是一个全球社区创作的窗外风景分享平台。用户可以录制并上传自己窗外的真实风景，也可以随机"打开"别人家的窗户，静静欣赏——巴黎的雨天、东京的夜景、或者某个不知名小村庄的田野。极简设计去掉了所有干扰，只留下窗户和外面的世界。
- **目标用户**: 需要数字逃离的城市人、旅行爱好者、随机探索者、想要放松的任何人
- **视觉风格**: 极简，几乎只有窗口内容本身，干净中性的界面框架
- **主色调**: 中性灰白、窗外自然色彩
- **氛围关键词**: 平静、漫游、治愈、随机探索、数字窗口
- **可归类方向**:
  - 常规: 放松工具 / 沉浸体验
  - 创意: 数字旅行 / 全球窗景共享
