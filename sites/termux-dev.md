# Termux

- **URL**: https://termux.dev
- **一句话**: Android 终端模拟器和 Linux 环境
- **详细说明**: Termux 是 Android 上的终端模拟器和 Linux 环境模拟器。无需 root 权限，即可在手机上运行 Linux 命令和工具。支持包管理器、编程环境、系统工具等，让手机成为强大的开发或命令行工作环境。
- **目标用户**: 安卓开发者、系统管理员、Linux 爱好者、移动设备开发者
- **视觉风格**: 极客风格，命令行界面
- **主色调**: 终端经典配色
- **氛围关键词**: 终端、Linux、Android、开发、开源
- **可归类方向**:
  - 常规: 开发工具、终端、系统工具
  - 创意: 移动开发环境、极客工具
