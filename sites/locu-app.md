# Locu

- **URL**: https://locu.app
- **一句话**: 开发者专用专注计时 + 任务管理 + 笔记一体工具，90 分钟深度工作制
- **详细说明**: 为开发者设计的专注工作工具，融合番茄钟（90 分钟深度工作）、任务跟踪和笔记功能，支持与 Jira/Linear 同步。
- **目标用户**: 开发者、需要深度工作的知识工作者
- **视觉风格**: 简洁，开发者审美
- **主色调**: 深色系
- **氛围关键词**: 专注、深度工作、开发者、效率、任务管理
- **可归类方向**:
  - 常规: 效率工具、开发者工具
  - 创意: 独立开发者后台
