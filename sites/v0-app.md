# v0

- **URL**: https://v0.app
- **一句话**: Vercel AI UI 生成器，自然语言生成 React 组件
- **详细说明**: v0 by Vercel 是 AI 驱动的代码生成平台，用于构建"Agents, Apps, and Websites"。通过自然语言提示生成可工作的应用程序，一键部署到 Vercel 生产环境。提供可视化设计模式(Design Mode)进行细节调整、GitHub 代码库同步功能、多种第三方 API 和工具集成。理念是"Prompt. Build. Publish."——从想法到上线仅需几分钟。Agentic by default，AI 自动规划、创建任务、连接数据库。
- **目标用户**: 开发者和设计师、需要快速原型开发的团队、学生(有专门的 Students 页面)、企业用户(Enterprise 版本)
- **视觉风格**: 现代极简主义，支持深色/浅色主题切换，深色模式使用 hsl(0 0% 4%)，浅色使用 hsl(0 0% 98%)。干净的界面布局，卡片式模板展示，基于 shadcn/ui 组件库。
- **主色调**: 深色/浅色双主题
- **氛围关键词**: 快速、智能、现代、Vercel 品质、AI 原生
- **可归类方向**:
  - 常规：AI 工具、代码生成、开发工具、UI 工具、Vercel 生态
  - 创意：提示即创造、Agentic 设计、从想法到上线、设计系统、模板库
