# Silicon Friendly

- **URL**: https://siliconfriendly.com
- **一句话**: 为网站打 AI 代理友好度评分（L0-L5），帮助开发者找到不会"拒绝"代理的网站
- **详细说明**: Silicon Friendly 是一个众包建立的 AI 代理网站目录，用30项标准将网站从 L0（未评级）到 L5（支持自主操作）分级，评估维度包括是否有 MCP 服务器、API 写权限、代理认证、webhook 支持等。开发者可以提交网站、声明自己的网站并帮助验证，代理可通过 API 获取 bearer token 搜索"对自己友好"的网站。
- **目标用户**: AI 开发者、构建 AI 代理的工程师、希望让网站对 AI 友好的产品团队
- **视觉风格**: 极简浅色调，信息密度高，功能型排版
- **主色调**: 浅色系，以内容信息为主
- **氛围关键词**: 开放标准、代理就绪、开发者向、众包验证、未来互联网基础设施
- **可归类方向**:
  - 常规: AI 工具 / 开发者资源
  - 创意: 互联网的代理通行证系统
