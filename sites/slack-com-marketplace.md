# Slack App Directory

- **URL**: https://slack.com/marketplace
- **一句话**: Slack 官方应用市场
- **详细说明**: Slack 官方应用和集成市场。提供数千个应用和集成工具，帮助用户扩展 Slack 的功能。支持按类别浏览、搜索和安装，是 Slack 生态的核心应用商店。
- **目标用户**: Slack 工作空间管理员、团队成员、寻求工作流优化的企业
- **视觉风格**: 企业级，官方商店风格
- **主色调**: Slack 品牌色（紫色）
- **氛围关键词**: 应用、集成、市场、生态、扩展
- **可归类方向**:
  - 常规: 应用市场、Slack 生态、工作流工具
  - 创意: 生产力生态、集成平台
