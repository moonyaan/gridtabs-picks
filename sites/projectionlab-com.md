# ProjectionLab

- **URL**: https://projectionlab.com
- **一句话**: 可交互的财务独立模拟器，用蒙特卡洛模拟和 Sankey 图预测你的整个人生财务轨迹
- **详细说明**: ProjectionLab 是一个隐私优先的财务规划工具，专门为追求财务独立/提早退休的人设计。支持蒙特卡洛成功率模拟、Sankey 现金流图、税务分析、遗产规划和历史回测。完全不连接真实账户，所有计算本地进行，支持全球多货币场景，是个人财务规划的"沙盘推演"神器。
- **目标用户**: 追求财务独立/提早退休的人（FIRE 社区）、个人理财爱好者、财务顾问
- **视觉风格**: 现代专业，数据可视化为核心，渐变背景配清晰图表
- **主色调**: 蓝绿渐变、深色/浅色主题皆支持
- **氛围关键词**: 财务规划、数据驱动、未来可视化、FIRE、隐私优先
- **可归类方向**:
  - 常规: 财务工具 / 退休规划
  - 创意: 人生财务沙盘 / 时间线可视化
