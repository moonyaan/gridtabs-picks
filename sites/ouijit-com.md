# Ouijit

- **URL**: https://ouijit.com
- **一句话**: 开源的 AI 代理任务与终端会话管理器，支持 git 工作树隔离、Kanban 看板和 VM 沙箱。
- **详细说明**: Ouijit 是专为 AI 代理开发工作流设计的可定制任务与终端管理器，支持 Claude Code、Codex 等多种代理 CLI。它提供每任务独立的 git 工作树（Copy-on-Write 克隆）、带生命周期钩子的 Kanban 看板、集成终端会话，以及通过 Lima 实现的 VM 沙箱隔离执行环境。开源免费，适合同时管理多个 AI 代理工作流的开发者。
- **目标用户**: 使用 AI 编程代理的开发者、需要管理并行 AI 任务的工程师
- **视觉风格**: 技术感强，现代看板布局，深色配色为主，终端风格 UI
- **主色调**: 深色/中性调色板，最小化强调色
- **氛围关键词**: 开源、高效、开发者中心、安全隔离、代理工作流
- **可归类方向**:
  - 常规: 开发者工具/AI代理管理
  - 创意: AI 代理军团的调度指挥台
