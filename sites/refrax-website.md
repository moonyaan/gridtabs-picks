# Refrax

- **URL**: https://refrax.website
- **一句话**: macOS 原生 WebKit 浏览器，垂直标签页 + 隐私优先，由一位开发者独立构建
- **详细说明**: Refrax 是一款替代 Arc 浏览器的 macOS 原生浏览器，使用 WebKit 引擎开发，主打优雅设计与垂直标签栏布局。完全隐私优先，无追踪无广告，由独立开发者从零打造，是 Arc 关闭后开发者社区反响热烈的替代方案。
- **目标用户**: macOS 用户、追求简洁和隐私的开发者、Arc 浏览器用户
- **视觉风格**: 极简 macOS 原生风格，干净优雅
- **主色调**: 白色/浅灰，macOS 系统色调
- **氛围关键词**: 隐私、极简、macOS 原生、垂直标签、独立开发
- **可归类方向**:
  - 常规: 浏览器 / 生产力工具
  - 创意: 一个开发者重建了整个浏览器
