# Immich

- **URL**: https://immich.app
- **一句话**: 自托管照片管理，Google Photos 替代品
- **详细说明**: Immich 是自托管照片和视频管理服务，支持移动端(Android/iOS)、服务器端应用及 API 接口。用户可上传、组织和访问个人媒体库。强调开源透明、跨平台支持(Android、iOS、服务器版本)、社区驱动(完整文档、API、数据集、路线图)、隐私优先(由 FUTO 支持，强调数据自有权)、官方商业支持。
- **目标用户**: 重视隐私和数据自主权的用户，技术爱好者，自建服务倾向者
- **视觉风格**: 现代极简设计，支持亮色和深色主题自适应。简洁导航布局，强调内容展示而非装饰。
- **主色调**: 现代简洁色调
- **氛围关键词**: 隐私、开源、自托管、照片管理、数据自主
- **可归类方向**:
  - 常规：照片管理、自托管工具、隐私工具、开源软件
  - 创意：Google Photos 替代品、隐私摄影库、自主照片管理、开源精神
