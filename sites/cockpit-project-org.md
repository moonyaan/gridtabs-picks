# Cockpit

- **URL**: https://cockpit-project.org/
- **一句话**: 浏览器里管理 Linux 服务器的图形化界面，点击鼠标完成磁盘/网络/容器/日志等所有操作
- **详细说明**: Cockpit 是一个开源的服务器 Web 图形管理界面，安装后通过浏览器即可进行原本需要命令行的服务器操作，包括网络配置、存储管理、虚拟机管理、容器控制和日志查看，还内置了终端。对 Linux 新手和 Windows 管理员非常友好，不影响现有命令行工作流，可与其他工具无缝切换。
- **目标用户**: Linux 服务器管理员（新手和老手）、Windows 背景的系统管理员、homelab 用户
- **视觉风格**: 简洁现代，仪表盘式布局，标签页导航，模态对话框
- **主色调**: 深色背景 + 白色文字，红色（Red Hat 品牌影响），蓝色交互元素
- **氛围关键词**: 实用、亲切、开放、高效、开源
- **可归类方向**:
  - 常规: 服务器管理工具
  - 创意: 自托管神器
