# All The Views

- **URL**: https://alltheviews.world
- **一句话**: 用算法找出地球上所有最长视线，45亿条可探索视角的交互式地图
- **详细说明**: All The Views 通过自定义算法 CacheTVS 计算地球上任意两点间的视线可达性，识别出最长的视线（目前记录为昆仑山脉到 Pik Dankova 的 530km）。提供约 45 亿条可探索视线的交互式地图和全球最长视角排行榜，将地理科学变成一场视觉探索之旅。
- **目标用户**: 地理爱好者、户外探险者、科学研究者、对地球感到好奇的任何人
- **视觉风格**: 简洁干净，浅/深色主题可切换，以高质量地景摄影为主视觉
- **主色调**: 中性白/深色背景、大地色系
- **氛围关键词**: 科学探索、震撼、数据驱动、地理奇迹、交互发现
- **可归类方向**:
  - 常规: 地理工具 / 数据可视化
  - 创意: 地球奇观 / 视角极限探索
