# Zapier

- **URL**: https://zapier.com
- **一句话**: 工作流自动化集成平台
- **详细说明**: Zapier 是无代码自动化平台，连接 8000+ 应用。提供 AI 工作流编排和 Agent 创建、数据库管理(Tables)、表单构建、聊天机器人部署、AI 工具与日常业务应用集成。面向企业(Nvidia、Airbnb、Meta、Disney、Mastercard)、中小型企业和初创公司。核心卖点是"Scale AI safely"(8000+ 集成快速部署)、时间节省(周省 20+ 小时，50 小时降至 3 小时)、收益影响(管道增加 100 万美元，节省 50 万+ 美元)、易用性(分钟内创建工作流无需编码)。
- **目标用户**: 企业、中小型企业、初创公司、RevOps/营销/销售/IT/HR/客户支持/行政团队
- **视觉风格**: 现代极简设计，橙色强调(#FF4F00)搭配奶油背景。自定义 Degular Display 排版，清晰网格布局、充裕留白
- **主色调**: 橙色强调、奶油背景
- **氛围关键词**: 自动化、无代码、AI 工作流、集成、企业级

  - 常规：自动化平台、无代码工具、工作流、AI 集成
