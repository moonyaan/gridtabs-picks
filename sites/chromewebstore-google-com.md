# Chrome Web Store

- **URL**: https://chromewebstore.google.com
- **一句话**: Chrome 扩展程序官方应用商店
- **详细说明**: Google Chrome 官方应用商店，提供海量扩展程序和应用下载。支持搜索、分类浏览、用户评分和评论。采用 Material Design 3 设计语言，支持亮色/暗色主题切换，使用 Roboto 字体。
- **目标用户**: Chrome 浏览器用户、寻求扩展功能的个人、应用开发者
- **视觉风格**: Material Design 3，支持亮暗主题，模态对话框，卡片式排列，动画过渡
- **主色调**: Material Design 系统色，亮暗主题自适应
- **氛围关键词**: 应用商店、扩展程序、官方平台、丰富生态、易用界面
- **可归类方向**:
  - 常规: 应用商店、Chrome 生态、工具发布
  - 创意: 产品发布渠道、开发者平台
