# Miro

- **URL**: https://miro.com
- **一句话**: 可视化协作白板平台，团队规划和创新工作空间

- **目标用户**: 从事设计、产品开发和战略规划的团队和组织
- **视觉风格**: 现代极简，清晰的排版和几何布局
- **主色调**: 蓝色为主 (#3859ff, #314cd9) + 黄色 (#fde050) + 青绿紫辅助色
- **氛围关键词**: 专业、创新、协作、技术前沿、创意、结构化而灵活
- **可归类方向**:
  - 常规: 协作工具、白板软件、项目管理、设计工具
  - 创意: 无限画布、思维可视化、团队大脑、创新实验室
