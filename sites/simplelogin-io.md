# SimpleLogin

- **URL**: https://simplelogin.io
- **一句话**: 开源邮箱别名服务，用匿名邮箱保护真实收件箱隐私
- **详细说明**: 创建无限邮箱别名，用于注册各种服务，邮件自动转发到真实邮箱。完全开源，已被 Proton 收购。有效防止垃圾邮件、数据泄露追踪和跨站追踪。支持浏览器扩展和移动端。
- **目标用户**: 注重隐私的用户、不想暴露真实邮箱的人
- **视觉风格**: 简洁现代，大量插画和图标
- **主色调**: 蓝色、白色
- **氛围关键词**: 隐私、信任、开源、简单
- **可归类方向**:
  - 常规: 隐私工具、邮箱管理
  - 创意: 给每个网站一个假身份
