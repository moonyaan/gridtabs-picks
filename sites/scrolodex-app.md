# Scrolodex

- **URL**: https://scrolodex.app
- **一句话**: macOS 轻量窗口切换器，按住 Option 键滚动鼠标即可在窗口间流畅跳转
- **详细说明**: Scrolodex 是一款极简 macOS 工具，让用户只需按住 ⌥ 键并滚动鼠标即可快速切换当前光标下的窗口，无需点击。支持 Dock 图标悬停切换、完全自定义快捷键，兼容 macOS 14+，操作体验极为流畅自然。
- **目标用户**: macOS 高级用户、开发者、追求高效窗口管理的键盘党
- **视觉风格**: 极简现代，图标化界面，单色/中性色调
- **主色调**: 单色灰白，深色背景
- **氛围关键词**: 高效、无摩擦、轻量、直觉、生产力
- **可归类方向**:
  - 常规: keyboard-people, focused-life
  - 创意: swiss-army-knife-sites
