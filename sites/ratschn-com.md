# Ratschn

- **URL**: https://ratschn.com/
- **一句话**: Mac 本地离线语音转文字工具，用 Rust + Tauri + CoreML 构建，完全不联网
- **详细说明**: Ratschn 是一款完全本地运行的 Mac 语音听写 App，不依赖云服务也不发送任何数据到外部。基于 Rust、Tauri 和 Apple CoreML 构建，在 macOS 上实现流畅的离线语音转文字。是对 Mac 自带听写功能和云端方案的隐私优先替代品。
- **目标用户**: 注重隐私的 Mac 用户，需要可靠离线语音输入的开发者和专业人士
- **视觉风格**: 极简，暗色调，技术感
- **主色调**: 深海军蓝/炭黑（#0D0D12），亮蓝（#0A84FF）
- **氛围关键词**: 隐私优先、离线、可靠、安全、专注
- **可归类方向**:
  - 常规: Mac 生产力工具
  - 创意: 隐私优先的本地工具
