# Pangolin

- **URL**: https://pangolin.net
- **一句话**: 开源自托管隧道代理平台，内网穿透 + 零信任访问控制，Cloudflare Tunnel 的自主替代品
- **详细说明**: Pangolin 提供内网穿透和反向代理能力，通过零信任网络架构让用户安全地将内网服务暴露到公网，同时保持完全的数据主权。作为 Cloudflare Tunnel 的开源替代方案，所有流量经过自己控制的服务器，适合对隐私和自主性有高要求的技术用户。
- **目标用户**: 自托管爱好者、Home Lab 玩家、需要安全内网穿透的开发者和小型团队
- **视觉风格**: 简洁技术文档风，功能导向，现代 CLI 工具美学
- **主色调**: 深色 + 橙色强调（穿山甲品牌色）
- **氛围关键词**: 自托管、内网穿透、零信任、开源、网络安全
- **可归类方向**:
  - 常规: 网络工具 / 自托管基础设施
  - 创意: 把服务器的控制权还给自己——开源对抗大厂隧道垄断
