# boringBar

- **URL**: https://boringbar.app
- **一句话**: macOS Dock 替代品，用 Windows 任务栏风格管理多桌面窗口，一目了然不混乱
- **详细说明**: boringBar 是一款 macOS 原生应用，用类似 Windows 任务栏的界面替代 Dock，支持按桌面分组显示窗口、即时预览和一键切换。针对管理多个项目和工作区的 Mac 用户，售价 $29.99 买断，以熟悉的方式解决 macOS 多窗口管理的困扰。
- **目标用户**: 多任务 Mac 用户、习惯 Windows 任务栏操作习惯的用户、多显示器工作者
- **视觉风格**: 简洁功能性，模仿 Windows 任务栏风格，深色系界面
- **主色调**: 深灰、白色、蓝色
- **氛围关键词**: productivity, macOS, window-management, keyboard, minimal
- **可归类方向**:
  - 常规: macOS 效率工具、窗口管理
  - 创意: keyboard-people、focused-life
