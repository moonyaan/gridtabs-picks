# WolfStack

- **URL**: https://wolfscale.org/
- **一句话**: 单个 Rust 二进制文件搞定所有服务器管理——VM、Docker、存储、网络一个仪表盘全覆盖
- **详细说明**: WolfStack 是一个 Rust 构建的通用服务器管理平台，安装在任意 Linux 系统上，自动适配环境，支持管理 Docker/LXC 容器、虚拟机、Proxmox 集群、存储和网络。提供多主题 Web 仪表盘，内置 WolfNet 加密网格网络连接跨数据中心服务器。Proxmox 的开源替代方案之一。
- **目标用户**: 服务器运维、系统管理员、独立开发者 homelab 用户、DevOps 工程师
- **视觉风格**: 现代 Web 仪表盘，支持多主题（暗黑、Midnight、玻璃质感、琥珀终端）
- **主色调**: 可切换多主题，以深色调为主
- **氛围关键词**: 统一、强大、开源、高效、工程师美学
- **可归类方向**:
  - 常规: 服务器基础设施管理
  - 创意: 自托管神器
