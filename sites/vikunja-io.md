# Vikunja

- **URL**: https://vikunja.io
- **一句话**: 开源任务管理，看板/甘特图/列表多视图
- **详细说明**: Vikunja 是开源、自托管的任务管理应用(AGPLv3 许可)。提供多视图模式(列表、看板、甘特图、表格)、项目层级结构及子项目、团队协作和任务分配、从 Todoist/Trello/Microsoft To-Do 迁移支持。核心卖点是"Privacy First"(开源代码透明)、灵活性(自托管或 Vikunja Cloud 托管)、EU-Based("Made and hosted in the EU")、成本友好(自托管免费)、用户导向(Bootstrapped 开发，优先用户而非风投)。
- **目标用户**: 隐私意识强的个人、需要团队协作的项目管理用户、从商业工具迁移的用户、自托管爱好者和企业
- **视觉风格**: 现代、清晰界面，支持明暗双主题。设计强调简洁，具有趣味品牌(草驼马/羊驼意象)。流畅动画、响应式布局、阴影效果、直观导航。
- **主色调**: 明暗双主题
- **氛围关键词**: 隐私、开源、自托管、多视图、EU 制造
- **可归类方向**:
  - 常规：任务管理、项目管理、开源软件、团队协作、自托管工具
  - 创意：多视图灵活性、隐私优先、EU 精神、羊驼品牌、反商业化
