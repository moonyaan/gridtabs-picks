# Raycast

- **URL**: https://www.raycast.com
- **一句话**: macOS 上的超级启动器和生产力中心，用命令栏替代传统应用切换
- **详细说明**: Raycast 是原生 macOS 应用，通过快捷键呼出命令栏，集成了应用启动、文件搜索、剪贴板管理、窗口控制、脚本执行等功能。相比 Alfred/Spotlight，Raycast 拥有更现代的界面设计、原生扩展商店、内置 AI 助手(Quick AI、AI Chat)，且对开发者极其友好，可以用 React 编写扩展。产品理念是"让专业用户的日常操作从点击变成打字"。
- **目标用户**: macOS 高级用户、开发者、追求效率的知识工作者、键盘流重度用户。背书用户包括 Guillermo Rauch (Vercel CEO)、MKBHD、Adam Wathan (Tailwind CSS) 等
- **视觉风格**: 现代简约、毛玻璃效果、深色模式优先、流畅动画、精致微交互、科技感十足。深色主题背景色 #070921 等深蓝黑色调，玻璃拟态效果(glass morphism)、渐变色彩。"Keyboard First"键盘优先的设计理念。
- **主色调**: 深蓝黑背景(#070921)、紫色/紫红渐变高亮、荧光色点缀
- **氛围关键词**: 专业、极速、克制、未来感、开发者友好
- **可归类方向**:
  - 常规：生产力工具、macOS 应用、启动器/Launcher、效率工具、开发者工具、AI 工具
  - 创意：紫色系网站、深色美学、键盘优先设计、命令行哲学、工作流中枢、专业主义、速度美学、毛玻璃美学
