# Tiny CV

- **URL**: https://tiny.cv
- **一句话**: 基于 Markdown 的极简一页简历生成器，实时预览纸质排版效果，支持在线分享和 PDF 导出
- **详细说明**: Tiny CV 提供左侧 Markdown 编辑、右侧实时纸张预览的双栏体验，强调"一页纸"的简历设计理念，帮助求职者快速生成专业简洁的简历并发布到公开链接或导出 PDF。界面专注无干扰，入门零学习成本。
- **目标用户**: 求职者、工程师、设计师、销售和产品经理等技术人才
- **视觉风格**: 极简现代双栏布局，专业编辑器风格
- **主色调**: 绿色系主题色（#065f46），深色主题支持
- **氛围关键词**: 专业、极简、聚焦、高效、无干扰
- **可归类方向**:
  - 常规: swiss-army-knife-sites, solo-founder-back-office
  - 创意: make-it-beautiful
