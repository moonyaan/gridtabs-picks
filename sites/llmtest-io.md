# LLMTest

- **URL**: https://llmtest.io
- **一句话**: AI API 代理层，自动追踪成本、优化 Prompt、在 LLM 提供商宕机时无缝切换备用模型
- **详细说明**: LLMTest 作为介于应用和 LLM 提供商之间的代理层，持续监控 AI 功能的实际表现，自动优化 prompt 并在后台通过"Autopilot"模式改善生产环境中的 AI 特性。理念是"快速上线，我们来打磨"，降低 AI 应用的运维成本。
- **目标用户**: AI 产品开发者、工程团队、正在生产环境部署 LLM 的初创公司
- **视觉风格**: 极简主义，明亮浅色主题配深色"机器模式"，单栏清晰布局
- **主色调**: 浅灰（#fafafa）配深黑文字（#111）
- **氛围关键词**: 效率、自动化、可靠、透明、实用
- **可归类方向**:
  - 常规: keyboard-people, swiss-army-knife-sites
  - 创意: solo-founder-back-office
