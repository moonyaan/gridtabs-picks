# GitHuman

- **URL**: https://githuman.dev/
- **一句话**: 在 git commit 前提供 GitHub 风格 diff 审查界面，让人类把关 AI 生成代码
- **详细说明**: AI 辅助编程时代的代码审查工具，在提交前展示清晰的 diff，让开发者在 AI 快速生成代码时保持对代码的把控力。
- **目标用户**: 使用 AI 编程工具的开发者、重视代码质量的工程师
- **视觉风格**: 开发者工具风格，GitHub 式界面
- **主色调**: 深色系
- **氛围关键词**: 代码审查、AI 时代、人机协作、质量把控
- **可归类方向**:
  - 常规: 开发者工具、AI 编程辅助
  - 创意: 人类监督 AI 的工具
