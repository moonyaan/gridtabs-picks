# Springfield Oracle

- **URL**: https://www.springfieldoracle.com/
- **一句话**: 辛普森一家每一条"预言"的来源、事件引用和事实核查数据库，有据可查无夸大
- **详细说明**: Springfield Oracle 是一个可搜索数据库，追踪《辛普森一家》中所有被称为"预言"的内容，每条记录都标注具体剧集来源和真实事件引证，诚实标注哪些属实、哪些被夸大。开源项目，支持社区投稿。不同于网络上充斥的不实盘点，这里有完整的核查链条。
- **目标用户**: 辛普森粉丝、流行文化爱好者、事实核查爱好者、媒体研究者
- **视觉风格**: 清晰信息展示型界面，数据驱动设计
- **主色调**: 以内容展示为主的中性配色
- **氛围关键词**: 严谨、有趣、怀旧、可信、事实核查精神
- **可归类方向**:
  - 常规: 流行文化数据库
  - 创意: 互联网奇趣博物馆
