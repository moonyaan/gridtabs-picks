# TapToyPia

- **URL**: https://memalign.github.io/m/taptoypia/index.html
- **一句话**: 浏览器里的星球殖民点击游戏，你是被派往新星球的先驱，一边探索动植物一边建立人类新家园。
- **详细说明**: TapToyPia 是一款轻量级浏览器点击游戏，玩家扮演被派往陌生星球的独立探索者，任务是建立可供人类定居的殖民地。游戏包含星球植物和动物的发现机制，以及逐步扩张的基地建设玩法。以点击游戏（clicker）为核心机制，节奏轻松，适合碎片时间消遣，无需安装，打开浏览器即玩。
- **目标用户**: 喜欢轻量放置类和点击游戏的玩家、想在休息时间轻松娱乐的用户
- **视觉风格**: 轻量化像素/卡通风格，浏览器原生界面
- **主色调**: 自然色调（星球探索主题）
- **氛围关键词**: 探索、轻松、放置、科幻、独立游戏
- **可归类方向**:
  - 常规: 浏览器游戏/休闲娱乐
  - 创意: 60秒开始一段跨星际的轻冒险
