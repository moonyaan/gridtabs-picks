# Neal.fun

- **URL**: https://neal.fun/internet-artifacts
- **一句话**: 互联网历史博物馆，用交互式时间线展示从互联网诞生到今天的标志性事件。
- **详细说明**: Neal Agarwal 的创意项目之一，以可滚动时间线形式收集展示互联网发展历程中的重要里程碑。每个 artifact 配有故事和交互细节。Neal.fun 整个站以创意互动实验闻名，还有 The Deep Sea、The Password Game 等病毒式传播的项目。
- **目标用户**: 互联网文化爱好者、科技历史迷、喜欢创意交互体验的任何人
- **视觉风格**: 极简白底，卡片网格布局，hover 微妙缩放和阴影效果，克制但充满童趣。首页有互动植物彩蛋。
- **主色调**: 白色、黑色、浅灰
- **氛围关键词**: 童趣、怀旧、探索、互联网考古、令人会心一笑
- **可归类方向**:
  - 常规：互动体验、互联网文化、创意项目、教育
  - 创意：时光机器、数字考古、互联网的童年、探索精神、独立开发者的浪漫、博物馆
