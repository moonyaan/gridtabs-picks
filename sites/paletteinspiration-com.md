# Palette Inspiration

- **URL**: https://paletteinspiration.com
- **一句话**: 从 2.3 万幅大师名画中提取配色方案，用艺术史驱动你的设计灵感
- **详细说明**: Palette Inspiration 收录了从文艺复兴到当代的 23,000+ 幅名画，自动提取每幅作品的色彩方案，供设计师直接借鉴。浏览大师如何搭配颜色，找到兼具历史厚度与审美品味的调色灵感。
- **目标用户**: 平面设计师、UI 设计师、插画师、艺术学生、视觉创意工作者
- **视觉风格**: 中性优雅，奶油色、大地色系，卡片式大图展示，博物馆质感
- **主色调**: 奶白色、大地色
- **氛围关键词**: 优雅、历史感、博物馆品质、精致、启发性
- **可归类方向**:
  - 常规: 设计灵感、配色工具
  - 创意: 让卢浮宫帮你选品牌色
