# Three-Body Problem — 3D

- **URL**: https://structuredlabs.github.io/threebodyproblem/
- **一句话**: 可交互的浏览器端 3D 三体问题物理模拟器，实时观察混沌轨道的动态演变
- **详细说明**: 直接在浏览器里运行的三维三体问题引力模拟，用户可以调整参数实时观察多个天体之间的混沌轨道行为。深色宇宙感界面配合流畅的 3D 渲染，兼具科学性与视觉美感。
- **目标用户**: 物理爱好者、学生和教育工作者、对混沌理论和天体力学感兴趣的人
- **视觉风格**: 深色科技风，深海军蓝背景，青色/青绿色发光强调，现代字体
- **主色调**: 深海蓝黑 (#0a0a0f) + 青色荧光强调 (#00d4aa)
- **氛围关键词**: 宇宙感、科学、精密、互动、大脑体操
- **可归类方向**:
  - 常规: 物理模拟 / 教育科学可视化
  - 创意: internet-curiosity-museum / pixel-and-handmade
