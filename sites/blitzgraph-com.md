# BlitzGraph

- **URL**: https://blitzgraph.com
- **一句话**: AI 原生图数据库后端，用 JSON 查询替代 SQL，让 AI 代理直接操作图结构数据
- **详细说明**: BlitzGraph 是专为 AI 代理设计的图数据库后端，使用类型化 JSON 查询替代传统 SQL 和 ORM，支持 MCP 服务器直接接入。无需数据库迁移即可扩展数据模型，被称为"AI 代理的 Supabase"，解决了 LLM 驱动应用中关系数据建模的复杂性问题。
- **目标用户**: AI 应用开发者、构建 LLM 驱动产品的工程师、需要灵活图数据结构的初创团队
- **视觉风格**: 极简现代深色主题，等宽字体代码块，几何符号点缀，大量留白
- **主色调**: 深色、黑白系
- **氛围关键词**: 前沿、开发者优先、代理中心、务实、高效
- **可归类方向**:
  - 常规: AI 基础设施、数据库工具
  - 创意: 为 AI 代理量身打造的"无 SQL 数据库"
