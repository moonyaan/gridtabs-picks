# AgentDrive

- **URL**: https://getagentdrive.com
- **一句话**: 专为 AI 代理设计的持久化云存储，一个 API 调用让代理输出文件跨会话永久保存
- **详细说明**: AI 代理运行环境（E2B、ChatGPT、Manus）任务结束后文件即删除，AgentDrive 填补这个空白，提供 REST API 和 MCP 工具让代理保存报告、代码、数据集，人类通过仪表盘查看结果。免费层提供 1GB 存储和 10,000 次 API 调用/月。
- **目标用户**: AI 工程师、构建多步骤代理工作流的开发者、部署代理舰队的团队
- **视觉风格**: 极简现代，大量留白，强视觉层级，等宽字体用于代码内容
- **主色调**: 白色 + 浅灰 + 深灰 + 蓝色强调 (#2563eb)
- **氛围关键词**: 专业、技术、可靠、轻量基础设施、开发者友好
- **可归类方向**:
  - 常规: AI 基础设施、开发者工具
  - 创意: 给 AI 代理的"云盘"
