# OpenWorkers

- **URL**: https://openworkers.com
- **一句话**: 开源自托管 Cloudflare Workers 运行时，用 V8 隔离在自己的基础设施上执行 JavaScript
- **详细说明**: OpenWorkers 是 Cloudflare Workers 编程模型的开源实现，让开发者在自有服务器上运行 JavaScript 边缘函数，无需依赖 Cloudflare。支持 KV 存储、数据库绑定和定时任务，适合需要无服务器计算但想保持数据本地化、避免供应商锁定的团队。
- **目标用户**: 后端开发者、需要自托管边缘计算的团队、关注供应商锁定风险的工程师
- **视觉风格**: 技术极简风，开发者文档风格
- **主色调**: 深色、橙色、白色
- **氛围关键词**: open-source, self-hosted, serverless, developer, freedom
- **可归类方向**:
  - 常规: 开发者工具、自托管
  - 创意: self-hosted-gems、keyboard-people
