# Publora

- **URL**: https://publora.com
- **一句话**: Agent 时代的发布 API，一次 API 调用同步发布到 X/LinkedIn 等 10+ 平台
- **详细说明**: 面向 AI Agent 和开发者的内容发布基础设施，单一 API 即可向所有主流社媒平台分发内容，简化多平台运营。
- **目标用户**: 开发者、内容创作者、AI Agent 构建者
- **视觉风格**: API 产品风格，简洁专业
- **主色调**: 深色，技术感
- **氛围关键词**: API、多平台发布、自动化、内容分发
- **可归类方向**:
  - 常规: 内容工具、开发者 API
  - 创意: Agent 时代的发布基础设施
