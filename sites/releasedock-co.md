# ReleaseDock

- **URL**: https://www.releasedock.co
- **一句话**: 产品发布管理平台，协调 changelog、发布节奏和团队沟通于一体
- **详细说明**: 专门解决产品发布流程中的协调痛点，将 changelog 管理、版本发布节奏规划和团队沟通整合在一个平台，帮助团队以更有组织的方式对外发布产品更新。
- **目标用户**: 产品经理、开发团队、SaaS 公司、注重发布节奏的团队
- **视觉风格**: 专业简洁，流程感强
- **主色调**: 白色/蓝色或绿色
- **氛围关键词**: 发布、changelog、协作、产品、节奏
- **可归类方向**:
  - 常规: 产品管理、团队协作
  - 创意: 让每次发布都值得纪念
