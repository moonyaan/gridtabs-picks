# Bench by Silverstream

- **URL**: https://bench.silverstream.ai
- **一句话**: 自动录制和存档每次 Claude Code 会话，让你随时回放、调试和分享整个编程过程
- **详细说明**: Bench 是专为 Claude Code 用户打造的会话记录工具，Mac/Linux 一分钟内完成配置后，它会静默记录所有 Claude Code 会话，包括每次工具调用和文件变更。你可以检查哪里出了问题，生成分享链接发给同事，或者把成功的会话历史嵌入 PR 说明中。完全免费。
- **目标用户**: 使用 Claude Code 的开发者、工程团队、需要 AI 编程工作流可视化的技术人员
- **视觉风格**: 简洁功能型，以信息展示为主
- **主色调**: 浅色系，功能导向设计
- **氛围关键词**: 透明、调试友好、团队协作、AI 工作流、开发者工具、免费
- **可归类方向**:
  - 常规: 开发者工具 / AI 编程辅助
  - 创意: AI 的黑匣子记录仪
