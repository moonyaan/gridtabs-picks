# WebAIM — AIM in the Browser

- **URL**: https://www.webaim.xyz
- **一句话**: 在浏览器里重现 2000 年代 AOL 即时通讯的完整体验
- **详细说明**: WebAIM 是一封写给早期互联网的情书，在现代浏览器里完整复刻了 AOL Instant Messenger（AIM）的经典功能：实时聊天、好友列表、离开状态消息、聊天室，甚至 Windows 98 风格的桌面图标和系统菜单音效。所有东西都可以在浏览器里运行，无需安装任何软件。
- **目标用户**: 互联网怀旧爱好者、对早期数字文化感兴趣的人、创意玩家
- **视觉风格**: Windows 98 / 2000 年代 UI 复古风格，像素边框和经典蓝色
- **主色调**: Windows 灰蓝 / 白色
- **氛围关键词**: 怀旧、复古、互联网历史、情怀、数字博物馆
- **可归类方向**:
  - 常规: internet-curiosity-museum
  - 创意: no-algorithm-sites、pixel-and-handmade
