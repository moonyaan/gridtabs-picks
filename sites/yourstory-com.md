# YourStory

- **URL**: https://yourstory.com
- **一句话**: 展示和分享创业故事的平台
- **详细说明**: YourStory 是印度和亚洲创业生态的主要媒体平台。展示创业故事、企业家人物、初创融资等内容。为创业者提供曝光机会，为投资人提供项目信息，促进创业生态发展。
- **目标用户**: 创业者和创始人、投资人、科技爱好者、媒体和从业者
- **视觉风格**: 新闻媒体风格，现代设计
- **主色调**: 媒体品牌色
- **氛围关键词**: 创业、故事、生态、亚洲、社区
- **可归类方向**:
  - 常规: 创业媒体、社区平台、资讯发布
  - 创意: 创业生态、故事传播、创业社区
