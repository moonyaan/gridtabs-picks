# Flowstep

- **URL**: https://flowstep.ai
- **一句话**: AI 驱动的 UI 设计工具，输入文字描述即可生成可编辑的生产级 React 界面
- **详细说明**: Flowstep 让设计师和产品团队用自然语言描述界面，直接生成完整可编辑的 UI，并导出 React/TypeScript/Tailwind 代码。支持实时协作，与 Figma 双向同步，跳过从灵感到设计稿的低效环节。
- **目标用户**: 设计师、产品经理、前端工程师
- **视觉风格**: 现代简洁，以实际 UI 示例展示能力，蓝色主色调，专业感强
- **主色调**: 蓝色
- **氛围关键词**: 设计效率、AI生成、协作、代码导出
- **可归类方向**:
  - 常规: AI 设计工具、UI/UX
  - 创意: ai-no-code-2026、make-it-beautiful
