# API to MCP

- **URL**: https://apitomcp.ai
- **一句话**: 把任意 REST/GraphQL API 一键转成托管 MCP 服务器，让 Claude、Cursor 等 AI 直接调用
- **详细说明**: API to MCP 提供可视化界面和 AI 代理构建器，将 REST、GraphQL 和 SaaS API 转换为托管的远程 HTTP MCP 服务器。支持 OAuth、API Key、Bearer Token 等多种认证方式，内置 JMESPath 响应映射，让 AI 工具能安全访问企业系统和第三方服务。
- **目标用户**: 想让 AI 代理访问内部系统的开发者、MCP 生态建设者、企业技术团队
- **视觉风格**: 开发者工具风格，功能明确，仪表盘布局
- **主色调**: 深色背景、紫色强调、白色文字
- **氛围关键词**: developer, MCP, API, AI-agent, integration
- **可归类方向**:
  - 常规: 开发者工具、AI 基础设施
  - 创意: swiss-army-knife-sites、keyboard-people
