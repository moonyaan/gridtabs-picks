# Textile

- **URL**: https://www.gettextile.app
- **一句话**: macOS 桌面应用，让你像织布一样拼接、变换和重用文本片段，支持键盘快捷键和命令执行。
- **详细说明**: Textile 是一款 macOS 桌面应用，专为需要频繁组合和复用文本的用户设计。它支持追加、前置、替换等文字变换操作，可触发系统命令，并与剪贴板无缝集成。所有文本片段以本地纯文本文件存储，完全离线、隐私优先，无云端同步。对于重度文字工作者、开发者和写作者来说是一个高效的文本积木工具。
- **目标用户**: 需要频繁重用文本片段的开发者、写作者、系统管理员和内容创作者
- **视觉风格**: 极简整洁，macOS 原生美感，蓝色强调色，清晰排版
- **主色调**: 蓝色 (#3b82f6)、白色背景、锌灰色
- **氛围关键词**: 高效、私密、自动化友好、本地优先、开发者向
- **可归类方向**:
  - 常规: 写作/文字处理工具
  - 创意: 把文字当积木，随心所欲拼出想要的表达
