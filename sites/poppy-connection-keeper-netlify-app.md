# Poppy — Connection Keeper

- **URL**: https://poppy-connection-keeper.netlify.app
- **一句话**: 用花园比喻管理人际关系的轻量提醒工具，帮你不忘联系重要的人
- **详细说明**: Poppy 把你的联系人变成一座需要浇水的花园，设置每个人的联系频率（每周/月/季度），到期时给你温和无压力的提醒。所有数据本地存储，完全免费，不订阅、不收集数据，专为有心维护人际关系却容易忘事的人设计。
- **目标用户**: 忙碌的社交人士、远距离保持联系的人、希望有意识维护友谊的用户
- **视觉风格**: 植物花园主题，柔和自然色调，手绘植物插图，温暖低压力的界面设计
- **主色调**: 绿色系、泥土色调、柔和自然色
- **氛围关键词**: 温柔、无压、自然、本地优先、无订阅、关怀
- **可归类方向**:
  - 常规: 个人 CRM / 关系管理工具
  - 创意: focused-life / pixel-and-handmade
