# Brie

- **URL**: https://brie.io
- **一句话**: 一键生成包含截图、会话回放、控制台错误和网络数据的完整 bug 报告，分享一个链接搞定
- **详细说明**: Brie 是浏览器扩展，点击即可自动捕获 bug 的完整现场证据——截图、会话录制、控制台日志、网络请求——并将一切打包成可分享的单一链接。平台自动去重、按影响排序，甚至能为常见 bug 起草修复 PR，直接接入 GitHub、Linear、Jira。
- **目标用户**: 开发团队、QA 工程师、需要高效提交 bug 的产品人员
- **视觉风格**: 简洁现代科技风，强调证据流和清晰度
- **主色调**: 白色、中性色系
- **氛围关键词**: 高效、协作、自动化、精准、省时
- **可归类方向**:
  - 常规: 开发者工具、Bug 追踪
  - 创意: 把 bug 截图时代送进历史的一键报告工具
