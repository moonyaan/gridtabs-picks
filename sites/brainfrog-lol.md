# BrainFrog

- **URL**: https://brainfrog.lol
- **一句话**: 脑洞大开的益智谜题游戏，每天一题，挑战你的思维边界
- **详细说明**: 每天推出一道需要跳出常规思维的益智谜题，谜题设计新奇有趣，往往需要出其不意的思路才能解开。适合喜欢挑战大脑、享受解谜乐趣的用户每日打卡。
- **目标用户**: 益智游戏爱好者、喜欢脑筋急转弯的人、每日打卡用户
- **视觉风格**: 趣味卡通风，轻松幽默
- **主色调**: 绿色/明亮配色
- **氛围关键词**: 益智、脑洞、每日、趣味、挑战
- **可归类方向**:
  - 常规: 游戏、益智
  - 创意: 每天给大脑做一道体操
