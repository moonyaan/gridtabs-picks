# Wario Synthesis Engine

- **URL**: https://wario.style
- **一句话**: 把任意 MIDI 文件转成 Game Boy 8-bit 芯片音乐，全程在浏览器内完成
- **详细说明**: Wario Synthesis Engine 使用 Web Audio API 将 MIDI 文件重新合成为 Game Boy 风格的芯片音乐（chiptune）。支持搜索已有曲目或上传自定义 MIDI 文件，生成后可直接导出为音频或 MIDI 格式。纯公益非商业项目，作者形容"just for lols"。
- **目标用户**: 复古游戏音乐爱好者、chiptune 粉丝、音乐创作者、怀旧一代
- **视觉风格**: Game Boy 掌机像素风，Wario 角色元素，极简复古界面
- **主色调**: 浅绿色、黑色（Game Boy 调色板）
- **氛围关键词**: 复古、怀旧、俏皮、创意、实验性
- **可归类方向**:
  - 常规: 音乐工具、格式转换
  - 创意: 把现代音乐送回 1989 年的时光机
