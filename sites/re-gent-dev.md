# Re_gent

- **URL**: https://re-gent.dev
- **一句话**: AI 编程代理的版本控制系统，追踪代理对话和代码改动，支持回滚、溯源和会话重放
- **详细说明**: Re_gent 为 AI 代理带来类似 Git 的历史管理能力，用 `rgt log`、`rgt blame`、`rgt revert` 等命令管理代理行为轨迹。当 AI 代理出错或产生意外改动时，开发者可以精准定位并回滚，而不是丢失整个会话上下文。
- **目标用户**: 重度使用 AI 编程代理的开发者、工程团队
- **视觉风格**: 极简技术风，深色/中性调色，CLI 命令展示为核心内容
- **主色调**: 深色/中性
- **氛围关键词**: AI代理版本控制、回滚、溯源、开发者工具
- **可归类方向**:
  - 常规: 开发者工具、AI代理管理
  - 创意: keyboard-people、self-hosted-gems
