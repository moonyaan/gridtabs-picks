# Hive Index

- **URL**: https://thehiveindex.com
- **一句话**: 收录 4100+ 在线社区的精选目录，按话题和平台分类浏览
- **详细说明**: Hive Index 覆盖 229 个话题、36 个平台，帮助用户发现感兴趣的专业、创意或娱乐社区。每个社区标注论坛、聊天、活动、课程等功能特征，方便快速筛选适合自己的圈子。
- **目标用户**: 专业人士、创业者、学生，任何想寻找同类圈子的人
- **视觉风格**: 干净现代，浅灰背景，卡片式布局，玫瑰红 / 黄色点缀
- **主色调**: 浅灰 + 玫瑰红 (#e11d48) + 黄色高亮
- **氛围关键词**: 发现、社区感、包容、专业、有序
- **可归类方向**:
  - 常规: community-tools / directories
  - 创意: weekend-websites-worth-opening
