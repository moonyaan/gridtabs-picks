# Mobbin

- **URL**: https://mobbin.com
- **一句话**: 真实 UI 设计模式库
- **详细说明**: Mobbin 是综合设计参考库，展示"1000+ iOS & Web 应用、200 个网站"，包含"589900 个屏幕"和"313400 个流程"。用户可搜索跨多类别的设计模式(屏幕、UI 元素、流程、文本)。支持 Figma 集成、交互式流程演示、原型模式模式，包含来自 ChatGPT、Airbnb、Nike、Dropbox 等大品牌的真实示例。提供高级搜索、"复制到 Figma"能力、交互式热点、每周更新。
- **目标用户**: 产品设计师和设计团队、UX/UI 专业人士、构建移动和网络应用的公司、寻求设计最佳实践的组织
- **视觉风格**: 现代清晰界面，支持亮色/深色主题。强调可访问性和组织，通过高级筛选能力和直观导航
- **主色调**: 现代设计风格
- **氛围关键词**: 设计灵感、真实示例、UI 模式、设计参考、Figma 集成
- **可归类方向**:
  - 常规：设计工具、设计参考、UI/UX、设计灵感
  - 创意：真实 UI 模式库、设计加速器、Figma 魔法、品牌设计研究
