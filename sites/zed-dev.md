# Zed

- **URL**: https://zed.dev
- **一句话**: Rust 构建的极速代码编辑器，原生支持多人实时协作和 AI 辅助编程
- **详细说明**: Zed 是由 Atom 和 Tree-sitter 原班人马打造的下一代代码编辑器，用 Rust 从零构建，速度极快。支持 AI 代理编辑、内联助手、多人实时协作（像共享文档一样写代码）、原生 Git 集成和扩展生态，提供亮色/暗色主题。
- **目标用户**: 追求速度与效率的专业开发者、重视 AI 辅助编程工作流的工程师
- **视觉风格**: 极简主义，网格线装饰，几何图案背景，支持亮/暗主题切换
- **主色调**: 蓝色为主色，米白/奶油色背景，深灰用于暗色模式
- **氛围关键词**: 极速、智能、协作、工匠精神、开源、前沿
- **可归类方向**:
  - 常规: 代码编辑器、开发者工具
  - 创意: 协作编程新范式
