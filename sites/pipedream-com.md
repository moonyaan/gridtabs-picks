# Pipedream

- **URL**: https://pipedream.com
- **一句话**: Zapier 替代品，工作流自动化
- **详细说明**: Pipedream 是连接 API、AI 系统、数据库和其他服务的平台。支持用户在不需要大量编码的情况下构建集成和自动化工作流。基于 Framer 构建，采用 Inter 字体族、组织化 CSS 架构、响应式设计(移动、平板、桌面)。面向寻求集成方案的开发者、自动化工作流的团队、连接 API 与 AI 和数据库的组织。定位为连接器，实现"APIs、AI、databases 等"之间的无缝通信。
- **目标用户**: 开发者、需要工作流自动化的团队、连接多个系统的组织
- **视觉风格**: 现代极简，基于 Framer 构建，Inter 字体族，响应式设计，专业科技美学，注重可读性
- **主色调**: 现代技术风格
- **氛围关键词**: API 连接、自动化、低代码、集成、开发友好
- **可归类方向**:
  - 常规：自动化平台、API 工具、工作流
  - 创意：Zapier 替代品、API 魔法、集成枢纽
