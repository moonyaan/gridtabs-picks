# Remainders

- **URL**: https://remainders.vercel.app
- **一句话**: 生命日历壁纸生成器，用点阵可视化你的人生周数
- **详细说明**: 基于 Memento Mori 理念，将人生可视化为 4160 个周的点阵图，生成手机锁屏壁纸。也支持年度进度追踪。让你每次解锁手机都能感受到时间的流逝，安静但有力。
- **目标用户**: 关注时间管理、追求有意识生活的人
- **视觉风格**: 极简，点阵美学，黑白灰为主
- **主色调**: 黑、白、灰
- **氛围关键词**: 沉思、极简、时间感知、哲学
- **可归类方向**:
  - 常规: 壁纸生成器、时间管理
  - 创意: 把人生装进一张壁纸
