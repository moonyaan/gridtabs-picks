# Paperless-ngx

- **URL**: https://paperless-ngx.com
- **一句话**: 把纸质文件扫描归档为可搜索数字档案的自托管文档管理系统
- **详细说明**: Paperless-ngx 是开源自托管文档管理系统，支持扫描仪接入、OCR 文字识别、自动分类打标签，将纸质文件变为可全文搜索的数字档案。支持 Docker 部署，Reddit r/selfhosted 社区中的常青推荐。
- **目标用户**: 自托管爱好者、需要管理大量纸质文件的家庭或中小企业用户
- **视觉风格**: 推断为实用简洁的管理系统风格
- **主色调**: 未知
- **氛围关键词**: 无纸化、自主掌控、全文搜索、开源、隐私优先
- **可归类方向**:
  - 常规: 文档管理、自托管、办公效率
  - 创意: 「把纸堆变成知识库」
