# How Browsers Work

- **URL**: https://howbrowserswork.com
- **一句话**: 交互式教程，用动画和代码演示浏览器从输入 URL 到渲染页面的全过程
- **详细说明**: 面向工程师和好奇心旺盛的人，用交互式动画逐步拆解浏览器工作原理：DNS 解析、TCP 连接、HTTP 请求、HTML 解析、DOM 构建、渲染管线。深入浅出，开源项目。
- **目标用户**: 前端工程师、对浏览器原理好奇的开发者
- **视觉风格**: 简洁极简，交互式动画和代码块
- **主色调**: 黑白为主、蓝色点缀
- **氛围关键词**: 教育、交互、技术、好奇心
- **可归类方向**:
  - 常规: 开发者教育、技术可视化
  - 创意: 打开浏览器的引擎盖
