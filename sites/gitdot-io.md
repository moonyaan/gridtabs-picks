# Gitdot

- **URL**: https://gitdot.io
- **一句话**: 开源 GitHub 替代品，用 Rust 构建，强调社区和每周建设挑战
- **详细说明**: Gitdot 是一个自托管友好的开源代码仓库托管平台，定位为"更好的 GitHub"。功能包括公开/私有仓库管理、用户和组织账号、代码趋势发现、版本 release 追踪，以及独特的"每周挑战"社区建设机制。目前仍在早期开发阶段（v0.1-v0.2），活跃招募贡献者。
- **目标用户**: 开源开发者、关注隐私和去中心化的工程师
- **视觉风格**: 简洁现代开发者风格，暗色主题
- **主色调**: 深色 / 橙色强调色
- **氛围关键词**: 开源、独立、社区驱动、去中心化、builder
- **可归类方向**:
  - 常规: keyboard-people、self-hosted-gems
  - 创意: swiss-army-knife-sites
