# Uselink

- **URL**: https://uselink.app
- **一句话**: 把任何文档转换成有视觉化编辑能力的可分享链接，内置 MCP 服务器供 AI 工具直接调用
- **详细说明**: Uselink 让用户将 HTML 或 Markdown 内容发布为可分享的可控链接，支持视觉化编辑而无需对方下载文件。平台内置 MCP 服务器集成，使 Claude、Cursor 等 AI 工具可以直接创建和管理链接。适合需要分发文档、产品说明或内容页面，同时希望保留编辑控制权的团队和个人。
- **目标用户**: 需要快速分发文档的开发者、产品团队、独立创作者
- **视觉风格**: 简洁现代，文档工具风格
- **主色调**: 白色为主，蓝色强调
- **氛围关键词**: 文档分享、MCP集成、AI工具、无需下载、快速发布
- **可归类方向**:
  - 常规: 文档分享工具 / 内容发布
  - 创意: swiss-army-knife-sites / solo-founder-back-office
