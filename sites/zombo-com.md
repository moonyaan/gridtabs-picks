# Zombo.com

- **URL**: https://zombo.com
- **一句话**: 互联网著名梗网站，庄重地用循环语音告诉你"你能做任何事"，然后什么都不发生
- **详细说明**: 自1999年上线以来，Zombo.com 只做一件事：播放一段循环的合成语音，用无比正式的语气宣告"欢迎来到 Zombo.com，在这里你能做任何事"。是互联网最持久的荒诞迷因之一，见证了整个 Web 时代的变迁。
- **目标用户**: 互联网老网民、迷因文化爱好者、所有曾经在网上漫游的人
- **视觉风格**: 极简黑色背景 + 旋转圆圈，刻意的90年代 Flash 遗风
- **主色调**: 黑色 + 白色
- **氛围关键词**: 迷因、荒诞、经典、循环、网络历史
- **可归类方向**:
  - 常规: 互联网文化 / 经典梗网站
  - 创意: 互联网的永动机——一个空洞承诺存活了25年
