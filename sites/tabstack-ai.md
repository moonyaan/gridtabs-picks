# Tabstack

- **URL**: https://tabstack.ai
- **一句话**: 让 AI 代理像人类一样点击、滚动、搜索网页的 Web 自动化研究 API，内置四大核心能力
- **详细说明**: Tabstack 提供 /Extract、/Generate、/Automate、/Research 四个 API 端点，让 AI 系统可以自主浏览网页、提取结构化数据并完成复杂的多步骤 Web 操作。平台遵守 robots.txt、用明确 user-agent 标识，在自动化的同时尊重发布者权益。适合构建 AI 代理的开发者，从免费个人用到企业级批量生产工作负载均有覆盖。
- **目标用户**: 构建 AI 代理的开发者，需要网页自动化和数据提取的工程团队
- **视觉风格**: 开发者工具风格，代码示例为主，简洁文档导向
- **主色调**: 深色背景，代码高亮色
- **氛围关键词**: AI代理、网页自动化、API、开发者工具、数据提取
- **可归类方向**:
  - 常规: AI 代理基础设施 / 开发者工具
  - 创意: swiss-army-knife-sites / keyboard-people
