# SessionCast

- **URL**: https://sessioncast.io
- **一句话**: PC 终端实时推流到手机，远程监控 AI 编程助手
- **详细说明**: CLI 工具，将桌面终端会话实时流式传输到移动设备。专为远程监控 Claude Code、Cursor 和 GitHub Copilot CLI 等 AI 编程助手设计。延迟仅 50ms，OAuth 安全认证，开源可信。
- **目标用户**: 需要远程工作灵活性的开发者，尤其是使用 AI CLI 工具、管理长构建和部署的程序员
- **视觉风格**: 现代极简技术界面，展示桌面到移动的分屏工作流
- **主色调**: 深色主题（终端美学）+ 蓝色交互元素 + 绿色状态指示
- **氛围关键词**: 实用高效、解放桌面、可靠安全、开源、AI 编程时代的生产力
- **可归类方向**:
  - 常规: 开发工具、远程工具、终端工具、生产力软件
  - 创意: 终端口袋化、移动监工、AI 助手遥控器、桌面自由
