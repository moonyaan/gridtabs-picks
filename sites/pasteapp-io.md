# Paste

- **URL**: https://pasteapp.io
- **一句话**: Mac 和 iPhone 上的隐私优先剪贴板管理器，无限历史记录、跨设备同步、敏感数据保护规则
- **详细说明**: Paste 保存所有复制过的内容并支持全文搜索，可以按任意顺序粘贴多个条目并保留格式。内置自定义规则过滤敏感数据（密码、卡号等），支持 Siri Shortcuts 和 iOS 键盘插件，让剪贴板历史随时触手可及。开发者、设计师和内容创作者可以用它管理代码片段、设计素材、邮件模板，告别反复寻找同一段文字的烦恼。
- **目标用户**: 频繁使用复制粘贴的 Mac/iPhone 用户，开发者，设计师，内容创作者
- **视觉风格**: macOS/iOS 原生设计语言，简洁卡片式布局
- **主色调**: 白色、紫色强调
- **氛围关键词**: 剪贴板、效率、隐私、跨设备、macOS工具
- **可归类方向**:
  - 常规: 效率工具 / 剪贴板管理
  - 创意: keyboard-people / swiss-army-knife-sites
