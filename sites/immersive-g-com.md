# Immersive Garden

- **URL**: https://immersive-g.com
- **一句話**: 法国顶级创意工作室的数字艺术展示站，每个项目都是一次沉浸式 WebGL 视觉旅程
- **详细说明**: Immersive Garden 是一家法国创意技术工作室，其官网本身即是作品集，集合了顶级 WebGL 交互体验和实验性数字艺术项目。画面精细、交互流畅，是 Web 视觉艺术的天花板级呈现。
- **目标用户**: 创意开发者、数字艺术爱好者、视觉设计师
- **视觉风格**: 沉浸式 3D WebGL，暗色系，粒子效果与流体动态并存
- **主色调**: 深黑 + 霓虹渐变
- **氛围关键词**: WebGL、数字艺术、沉浸式、创意技术、视觉震撼
- **可归类方向**:
  - 常规: 创意工作室 / 数字艺术
  - 创意: 浏览器能有多美——用代码建造的数字美术馆
