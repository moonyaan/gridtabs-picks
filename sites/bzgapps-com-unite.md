# Unite Pro

- **URL**: https://www.bzgapps.com/unite
- **一句话**: 把任意网站变成专属 Mac App，支持窗口/侧边栏/菜单栏三种模式，深度 macOS 集成
- **详细说明**: Unite Pro 是 Mac 上把网站转成原生 App 的工具，几秒内创建出有 Dock 图标、通知支持和独立 Cookie 的网站应用。支持三种运行模式（全屏窗口、持久侧边栏、轻量菜单栏），可自动将仅支持亮色的网站转成暗色模式，内置 AI 覆盖层和 Dock 监控预览，是 Fluid 和 Station 的有力替代品。
- **目标用户**: Mac 重度用户、需要将 Web App（Notion/Figma/Linear）原生化的生产力爱好者
- **视觉风格**: 现代简洁，macOS 原生质感，强调融入系统
- **主色调**: 白色/浅灰为主，蓝色（#116dff）强调
- **氛围关键词**: 原生感、高效、整洁、专注、Mac优先
- **可归类方向**:
  - 常规: Mac 生产力工具
  - 创意: 浏览器变身 App
