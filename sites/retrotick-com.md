# RetroTick

- **URL**: https://retrotick.com
- **一句话**: 在浏览器里直接运行经典 Windows 和 DOS 程序，无需安装任何东西
- **详细说明**: RetroTick 是一个基于浏览器的 x86 模拟器，让你可以直接在网页里运行几十年前的 Windows 和 DOS 应用程序。支持 PE/NE 可执行文件格式、FPU 指令集，以及键盘、文件 IO 和内存管理，完整复现经典软件体验。对复古计算爱好者、软件历史研究者和怀旧玩家来说是一个神奇的时间机器。
- **目标用户**: 复古计算爱好者、软件历史研究者、怀旧用户、开发者
- **视觉风格**: 极简功能主义，界面以运行的古典程序为主角，带有浓厚的复古科技感
- **主色调**: 黑色终端绿、经典灰、Windows 蓝
- **氛围关键词**: 怀旧、历史保存、复古科技、极客、时光机
- **可归类方向**:
  - 常规: 复古工具 / 软件模拟器
  - 创意: 数字博物馆 / 时光回溯
