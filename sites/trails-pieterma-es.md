# Trails

- **URL**: https://trails.pieterma.es
- **一句话**: AI 发现 100 部文学作品之间的主题关联，把书籍变成可漫游的思想网络
- **详细说明**: Trails 是一个 AI 驱动的书籍主题发现平台，自动分析跨作品的概念线索，生成如"有用的谎言"、"看不见的裂缝"等主题路径，将不同书籍以思想线索相连。由 Claude Code 自动处理，界面以封面缩略图网格呈现，让读者从主题而非作者发现下一本书。
- **目标用户**: 热爱文学的读者、喜欢跨书发现知识联系的思考者、书评爱好者
- **视觉风格**: 干净的图书馆风格，封面缩略图网格布局
- **主色调**: 米白、深棕、书封彩色
- **氛围关键词**: books, discovery, connections, literary, AI
- **可归类方向**:
  - 常规: 书籍推荐、知识发现
  - 创意: no-algorithm-sites、internet-curiosity-museum
