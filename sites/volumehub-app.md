# VolumeHub

- **URL**: https://volumehub.app
- **一句话**: macOS 独立应用音量控制工具，无需驱动
- **详细说明**: VolumeHub 是原生 macOS 应用，独立控制每个应用的音量(0-100%)，提供实时音频可视化仪表、一键切换音频输出设备、10 频段均衡器(含 17 种预设)。Focus Audio 功能可在通话时自动降低其他应用音量。提供三种视图模式(Compact/Comfort/Full)。零侵入设计：无内核扩展、无音频驱动，100% SwiftUI 构建，使用 Apple 原生 Audio Tap API。
- **目标用户**: macOS 用户(需 macOS 14.2 Sonoma 或更高)、需要精细音频控制的用户(如边听音乐边开会)、注重隐私的用户、专业用户与普通用户
- **视觉风格**: Native macOS Design，Built with SwiftUI，原生菜单栏弹窗界面。深色主题，简洁现代，符合 macOS 原生设计语言。极简、专业、功能性强。
- **主色调**: 深色主题
- **氛围关键词**: 原生、轻量、隐私、专业、零开销
- **可归类方向**:
  - 常规：macOS 应用、音频工具、生产力工具、系统增强
  - 创意：音量艺术、零侵入哲学、原生美学、隐私优先、SwiftUI 之美
