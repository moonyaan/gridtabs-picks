# Liso

- **URL**: https://liso.dev
- **一句话**: 零知识加密的密码管理器与 2FA 认证器，支持自建服务器完全掌控数据
- **详细说明**: Liso 是一款注重安全与隐私的密码管理 + 双因素认证应用，采用密码学加密确保即使服务商也无法读取用户数据。支持跨设备同步，可选官方托管或完全自托管，面向对隐私有高要求的用户。
- **目标用户**: 注重数字安全和隐私的个人用户、开发者、技术爱好者
- **视觉风格**: 专业简洁，安全感强
- **主色调**: 深色系为主
- **氛围关键词**: 隐私、安全、零知识、自托管、掌控
- **可归类方向**:
  - 常规: 密码安全 / 隐私工具
  - 创意: 自托管精品 / 数字自主权
