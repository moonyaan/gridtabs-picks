# Visual Design Tools That Ship

From screenshot polish to AI-generated presentations, from product demo videos to SVG backgrounds — these tools close the gap between idea and finished visual. No design degree required.

# 视觉设计工具：从想法到成品

截图美化、AI 演示文稿、产品演示视频、矢量背景……这 8 个工具覆盖了视觉创作链条的每一环。不需要设计背景，上手就能出活。

## Sites

### Screen Studio
- **URL**: https://screen.studio
- **EN**: macOS screen recorder built for product demos. Auto-zoom follows your cursor, backgrounds look like Apple keynote slides, and you pay once — no subscription. The recordings look nothing like screen recordings.
- **ZH**: macOS 专属录屏工具，自动缩放跟随光标，背景渲染有 Apple 发布会的质感，一次性买断。录出来的东西完全不像屏幕录像。

### Photoroom
- **URL**: https://www.photoroom.com
- **EN**: AI photo editor built for product images. Remove backgrounds, generate new scenes, place virtual models, batch-edit hundreds of shots. Claims to cut photography costs by 90% — and looking at the before/afters, that tracks.
- **ZH**: 专为产品图而生的 AI 图片编辑器。智能抠图、AI 背景生成、虚拟模特、批量处理，官方说能降低 90% 拍摄成本，看完前后对比基本信了。

### Pika
- **URL**: https://pika.style
- **EN**: Screenshot beautifier with real teeth — App Store mockups, code images, tweet screenshots, device frames, social media exports. Has a Chrome extension for instant capture and a VSCode extension for code shots. YC-backed.
- **ZH**: 截图美化工具，但功能比"美化"两字厚得多——App Store 展示图、代码截图、推文截图、设备框架、社交媒体尺寸。Chrome 扩展一键捕获，VSCode 扩展出代码图。YC 支持。

### Kodo
- **URL**: https://www.usekodo.ai
- **EN**: Describe what you want, Kodo generates editable layered designs — text, colors, layout, spacing all tweakable. Posters, slides, social images, menus. Outputs PNG/JPG/PDF/PPTX. The edit-everything part is what actually makes it useful.
- **ZH**: 描述想要什么，Kodo 生成可编辑的分层设计——文字、颜色、布局、间距全部可以直接改。海报、幻灯片、社交图片、菜单，可导出 PNG/JPG/PDF/PPTX。「可编辑」是它比其他 AI 设计工具多出来的那块。

### Chronicle
- **URL**: https://chroniclehq.com
- **EN**: AI presentation tool that first builds a narrative structure, then designs the slides around it. Templates made by designers from Apple, McKinsey, and IDEO. Real-time collaboration, PowerPoint/PDF export, no AI clichés.
- **ZH**: AI 演示工具，先帮你构建叙事结构（Storyline），再自动排版设计。模板来自 Apple、麦肯锡、IDEO 的设计师。实时协作，可导出 PowerPoint/PDF，没有那种一眼假的 AI 风格。

### Bazaar
- **URL**: https://bazaar.it
- **EN**: Describe your product in a sentence, Bazaar turns it into a polished demo video — pulls from Figma files, images, or YouTube links you provide. Built specifically for SaaS teams who need motion graphics without a motion designer.
- **ZH**: 一句话描述产品，Bazaar 自动合成演示视频——可以上传 Figma 文件、图片、YouTube 链接作为素材。专为没有动效设计师的 SaaS 团队而生。

### Cardboard
- **URL**: https://usecardboard.com
- **EN**: "Cursor for video editing" — type what you want to change, the AI does it. Search footage by what's in it, auto-captions, smart cuts. Runs in the browser, supports real-time team collaboration. YC-backed.
- **ZH**: "视频剪辑领域的 Cursor"——用文字说要做什么改动，AI 直接执行。按画面内容搜索素材，自动字幕，智能剪辑。完全跑在浏览器里，支持团队实时协作。YC 支持。

### SVG Backgrounds
- **URL**: https://www.svgbackgrounds.com/set/free-svg-backgrounds-and-patterns/
- **EN**: 48 free SVG backgrounds and patterns — geometric, gradient, textured, abstract. Customize colors, scale, animation, then copy the CSS or SVG code directly. Built by one person, completely free to use.
- **ZH**: 48 套免费 SVG 背景和图案，几何、渐变、纹理、抽象各种风格都有。自定义颜色、缩放、动画，直接复制 CSS 或 SVG 代码。一个人做的，完全免费。
