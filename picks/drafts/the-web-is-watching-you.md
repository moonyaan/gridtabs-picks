# Your Browser Has Been Busy

Eight sites about what the web knows, what it exposes, and what happens when someone looks back. Some of this is about you. Some is about everyone else. All of it is happening right now.

# 你以为自己在上网，其实网页在研究你

浏览器比你想象的知道得多——你的位置、电池电量、屏幕尺寸，甚至哪些文件暴露在外。这 8 个网站把那扇平时看不见的窗拉开。

## Sites

### Taken
- **URL**: https://sinceyouarrived.world/taken
- **EN**: Shows exactly what your browser hands over the moment you arrive — fonts, GPU, timezone, location — with no cookies or login required.
- **ZH**: 你还没点任何东西，浏览器就已经交出了你的字体列表、显卡信息、时区和位置。没有 Cookie，没有登录，只是浏览器默认就会泄露的那些数据。

### GeoSpoof
- **URL**: https://geospoof.com
- **EN**: Your location leaks through GPS, timezone, IP, and WebRTC — sometimes telling different stories, sometimes the same. GeoSpoof masks all of them at once.
- **ZH**: 你的地理位置会通过 GPS、时区、IP、WebRTC 泄露——可能说着不同的故事，也可能指向同一个地方。GeoSpoof 同时覆盖所有这些信号。

### Cerast Intelligence
- **URL**: https://search.cerast-intelligence.com/
- **EN**: Type any domain and see exposed files — config backups, admin panels, things left in plain sight. The web knows what people forgot to hide.
- **ZH**: 输入任意域名，看看什么东西被遗忘在那里没人关上门——配置文件、备份、本不该公开的管理面板。

### IsUpMap
- **URL**: https://isupmap.com
- **EN**: A real-time heatmap of 90+ internet services — who's up, who's struggling, who's down. This is the infrastructure pulse, made visible.
- **ZH**: 90 多个互联网服务的实时热力图——谁在线、谁在抖、谁挂了。这里把互联网的脉搏可视化。

### Refuse
- **URL**: https://refuse.dev/
- **EN**: Intercepts npm install and blocks packages with known vulnerabilities before they land in your project. The web's supply chain has cracks; this patches them first.
- **ZH**: 在 npm install 阶段拦截依赖链，把含有已知漏洞的包挡在项目门外。互联网的供应链有裂缝，这个工具在裂缝打开之前打上补丁。

### Is This Tech Dead
- **URL**: https://isthistechdead.com
- **EN**: Silverlight: 96.9% dead. Kubernetes: very much alive. Tracks the "Deaditude" (死亡指数) of frameworks and platforms — what the web has moved on from and what it's still running on.
- **ZH**: Silverlight：96.9% 死亡。Kubernetes：依然鲜活。这个网站追踪各种框架和平台的「死亡指数」(Deaditude)——记录互联网已经翻篇和还在用的那些技术。

### Signum
- **URL**: https://signumaiapp.com
- **EN**: Paste any domain, get a scam risk score from 15+ threat intelligence sources. As the web gets better at looking trustworthy, tools like this matter more.
- **ZH**: 把域名粘贴进去，基于 15 个以上威胁情报来源得到一个诈骗风险评分。随着网站越来越善于伪装，能看穿幕布背后的工具就越重要。

### Nobie
- **URL**: https://nobie.com
- **EN**: A spreadsheet app for Mac that runs entirely offline — your data never touches a server. In a collection about what the web sees, Nobie is the counterpoint: don't give it anything to see.
- **ZH**: 完全离线运行的 Mac 表格应用，数据从不经过任何服务器。在这个关于「网络看到了什么」的合集里，Nobie 是那个反问：如果你什么都不给它看，会怎样？
