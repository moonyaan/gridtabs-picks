# Tools for People Who Live at the Keyboard

Editors that think. Runtimes that sprint. A terminal orchestrator, a type trainer, a launcher that never makes you reach for the mouse. If your hands are the main interface — this one's for you.

# 键盘人的工具箱

给那些把键盘当第二皮肤的人。不是所有工具都需要鼠标，不是所有效率都来自界面——有些工具就是为键盘而生，用完就知道少它不行。

## Sites

### Zed
- **URL**: https://zed.dev
- **EN**: Built in Rust by the team behind Atom and Tree-sitter, Zed loads instantly and stays out of your way — until you need AI-assisted edits or real-time co-authoring, then it's right there. Speed as a design principle.
- **ZH**: Atom 原班人马用 Rust 重写的编辑器，打开就是工作状态，没有任何多余的仪式感。AI 编辑、多人实时协作都是原生支持，不是插件——这是他们认真思考"编辑器该长什么样"之后的答案。

### Cursor
- **URL**: https://cursor.com
- **EN**: An editor that knows your codebase. AI isn't bolted on — it's baked into every autocomplete, every refactor, every chat. The editor that finally makes pair programming feel like a natural state.
- **ZH**: 不是装了 Copilot 插件的 VSCode，而是从架构底层为 AI 协作重新设计的编辑器。你写代码，AI 理解上下文，两者之间几乎没有摩擦——这是很多人换过来就不想换回去的原因。

### Windsurf
- **URL**: https://windsurf.com
- **EN**: Codeium's take on the AI editor: Cascade, their agentic engine, handles the multi-step tasks while you keep the big picture. Less context-switching, more flow. A serious challenger to Cursor that's worth trying yourself.
- **ZH**: Codeium 做的 AI 编辑器，核心是 Cascade 代理引擎——它能理解你在做什么，主动帮你处理多步骤任务。不是问答式 AI，更像一个真的在帮你干活的搭档。

### Bun
- **URL**: https://bun.sh
- **EN**: JavaScript runtime, package manager, bundler, test runner — all in one binary, all blazing fast. Switching from Node feels like going from dial-up to fiber. The ecosystem is finally catching up to what Bun has been promising.
- **ZH**: 一个二进制文件搞定运行时、包管理、打包、测试——还比 Node.js 快到让人不敢相信。不是玩具项目，是很多团队正在悄悄迁移的选择，性能数字说话。

### Bruno
- **URL**: https://www.usebruno.com
- **EN**: API client with no account, no cloud sync, no subscription. Your collections live as plain text files in your repo, travel with your code in Git. The anti-Postman for people who think tools shouldn't own your data.
- **ZH**: API 请求文件存在本地，格式是纯文本，直接进 Git。没有账号、没有云、没有按月付费。如果你觉得 Postman 越来越重、越来越贵，Bruno 是那个诚实的替代品。

### Codelegate
- **URL**: https://codelegate.dev
- **EN**: Keyboard-driven TUI that runs multiple AI coding agents in parallel across the same codebase using Git worktrees. When you're orchestrating Claude Code or Codex CLI, this is the cockpit. Pure terminal, no nonsense.
- **ZH**: 纯终端界面，让多个 AI 编程代理（Claude Code、Codex CLI）在同一仓库里并行工作，互不冲突。全键盘操作，没有图形界面的开销。正在用 AI 写代码的人，这是专门为你们设计的调度层。

### MonoSketch
- **URL**: https://app.monosketch.io
- **EN**: Draw system diagrams and architecture maps in ASCII style, right in the browser. The output drops straight into code comments or docs without losing fidelity. For developers who've always treated diagrams as a form of writing.
- **ZH**: 浏览器里画 ASCII 风格的系统架构图，结果可以直接粘进代码注释或技术文档。给那些觉得"写图"比"画图"更自然的工程师——用键盘把系统说清楚。

### Keybr
- **URL**: https://www.keybr.com
- **EN**: Touch typing practice built around statistical learning — it tracks where you slow down and drills those specific keys until they feel automatic. Most keyboard people got fast by accident; Keybr gets you fast on purpose.
- **ZH**: 不是随机练习，是追踪你的弱项按键，针对性反复训练直到手指形成肌肉记忆。很多人打字快是因为练的时间长——Keybr 是认真设计过的方法，花更少时间达到同样效果。

### Raycast
- **URL**: https://www.raycast.com
- **EN**: The Mac launcher that made "keyboard first" feel like a product philosophy, not a power-user trick. One shortcut to search files, run scripts, control windows, talk to AI, or do anything in your 1,300+ extensions. Less mousing. More doing.
- **ZH**: 一个快捷键呼出，然后什么都能做：开应用、搜文件、控窗口、问 AI、跑脚本。Raycast 不只是效率工具，它改变了你和 Mac 交互的方式——从点击到打字，从被动选择到主动召唤。
